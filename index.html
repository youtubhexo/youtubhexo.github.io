# Python网络编程初探：让你的代码连接世界！🌍

想让你的Python程序也能像微信一样和服务器聊天吗？网络编程就是实现这个功能的关键！听起来是不是有点高大上？别怕，今天我们就来揭开它的神秘面纱！😉

# 网络编程三要素：IP地址、端口和协议

在网络世界里，要找到一台特定的计算机并与之通信，就像我们寄快递需要地址一样，需要以下三个要素：

  • IP地址 (IP Address)：就像每台电脑在网络上的唯一地址，用于标识网络上的设备。
  • 端口 (Port)：一台计算机上可以运行多个网络程序，端口号就像是这些程序的不同房间号，用于区分不同的网络服务。
  • 协议 (Protocol)：网络通信需要双方遵守共同的“语言”和规则，这就是协议。例如，我们平时上网常用的 HTTP 就是一种协议。

今天我们主要关注 TCP协议，它是一种面向连接、可靠的协议，就像打电话一样，需要先建立连接才能进行数据传输，并且能保证数据传输的可靠性。

# 字符串的编码与解码

在网络传输中，数据通常以字节流的形式进行。而我们平时处理的文本是字符串。所以，在进行网络传输前，我们需要将字符串编码成字节串，接收到数据后，再将字节串解码成字符串。

常用的编码格式是 UTF-8，它可以支持各种语言的字符。

# Socket编程初体验

在Python中，我们可以使用 socket 模块进行网络编程。socket 可以看作是网络通信的一个端点，通过它可以发送和接收数据。

简单来说，TCP网络编程通常涉及以下步骤：

  1. 创建Socket：就像创建一根网线插头。
  2. 绑定IP地址和端口 (服务器端)：告诉别人你的“地址”和“房间号”，这样别人才能找到你。
  3. 监听连接 (服务器端)：等待别人的“电话”打进来。
  4. 接受连接 (服务器端)：有人打来“电话”了，接听一下。
  5. 发送/接收数据：开始“聊天”。
  6. 关闭连接：结束“通话”，挂断“电话”。

# 普通版本 vs Pro版本？

在网络编程中，可能会有不同版本的实现方式，例如基础版本可能只实现最基本的功能，而 Pro 版本可能会增加更多高级特性，比如并发处理、更复杂的协议支持等等。这就像我们使用的App，可能会有免费版和付费版，付费版功能更多更强大。

  • 在进行网络编程时，要理解 IP 地址和端口的概念。
  • 数据在网络传输前需要进行编码，接收后需要进行解码，确保编码格式一致。
  • socket 模块是Python进行网络编程的核心工具。

# 告别崩溃！Python异常处理小技巧！🛡️

有没有遇到过程序突然报错停止的情况？那感觉简直太崩溃了！🤯 但是有了Python的异常处理机制，我们就可以优雅地捕获和处理这些错误，让程序即使遇到问题也能继续运行下去！是不是很神奇？✨

# 什么是异常？🤔

简单来说，异常就是在程序运行过程中发生的错误或意外情况。比如，除数为零啦，访问了不存在的列表索引啦，文件找不到啦等等。这些都会导致程序中断。

# try...except：捕获你的小错误！🎣

Python中使用 try...except 语句来捕获和处理异常。try 块中放置可能出错的代码，如果 try 块中的代码发生了异常，程序会跳转到对应的 except 块中执行处理代码。

  • 捕获单个异常

    上面的代码尝试进行除以零的操作，这会引发 ZeroDivisionError 异常。但是因为我们用了 try...except 捕获了这个异常，所以程序不会崩溃，而是会执行 except 块中的代码。

  • 捕获多个异常
    有时候一段代码可能会引发多种类型的异常，我们可以在一个 except 后面跟着多个异常类型，用逗号 , 分隔，或者使用多个 except 块来分别处理不同的异常。

    使用 as e 可以获取到具体的异常信息哦！

# else 和 finally：让你的异常处理更完整！🌟

除了 try 和 except，我们还可以使用 else 和 finally 块来完善异常处理。

  • else 块：当 try 块中的代码没有发生任何异常时，else 块中的代码会被执行。
  • finally 块：无论 try 块中的代码是否发生异常，finally 块中的代码都会被执行。通常用于进行清理操作，比如关闭文件、释放资源等。

# 异常的传递性 📢

如果一个函数内部发生了异常，并且没有被该函数处理，那么这个异常会向上层函数传递，直到被处理或者到达程序的顶层（导致程序崩溃）。

# 模块和包的导入

在Python中，我们可以通过 import 语句导入其他模块和包来使用别人写好的代码。这就像我们使用别人做好的工具一样，可以大大提高开发效率！

  • 导入模块：import 模块名
  • 导入包：import 包名.模块名

# 小贴士

  • 不要滥用异常处理，只捕获你期望处理的异常。
  • 尽量提供清晰的异常信息，方便调试。
  • finally 块非常重要，用于确保资源的正确释放。

08-编码与文件操作

Python文件读写全攻略！告别文件操作烦恼！📝

哈喽兄弟们！今天给大家分享的是Python文件操作和JSON数据处理的实用技巧！学会这些，让你的数据处理工作效率翻倍！再也不用对着文件发愁啦！🥳

# 文件操作初体验：打开、读取与关闭

在Python中，我们可以轻松地对文件进行读写操作。

  • 打开文件 open()：使用 open() 函数打开一个文件，你需要指定文件名和打开模式（例如 'r' 表示读取，'w' 表示写入）。还可以指定编码格式，推荐使用 'utf-8' 哦！
  • 读取文件内容 read()：使用 read() 方法可以一次性读取文件的全部内容，返回一个字符串。
  • 关闭文件 close()：操作完文件后，一定要记得使用 close() 方法关闭文件，释放系统资源！这是一个好习惯哦！✅

# 更优雅的文件操作：with open()

为了避免忘记关闭文件，Python提供了 with open() 语句，它会在代码块执行结束后自动关闭文件，超级方便！强烈推荐使用这种方式！👍

# 写入文件内容

  • 写入字符串 write()：使用 write() 方法可以将字符串写入文件中。如果文件不存在，会自动创建；如果文件已存在，会覆盖原有内容（使用 'w' 模式）。
  • 写入列表 writelines()：如果你想一次性写入多行文本，可以使用 writelines() 方法，传入一个包含所有行的列表。

# 逐行读取文件内容

有时候文件很大，一次性读取所有内容可能会占用大量内存。这时，我们可以逐行读取文件内容。

  • 使用 for 循环：可以直接对文件对象进行迭代，每次返回文件的一行内容。
  • 使用 readlines()：readlines() 方法会一次性读取所有行，并返回一个包含所有行的列表。如果文件很大，不推荐使用。

# JSON数据处理：让数据更有条理！✨

JSON（JavaScript Object Notation）是一种轻量级的数据交换格式，易于阅读和编写，也方便机器解析和生成。在Python中，我们可以使用 json 模块来处理JSON数据。

  • 导入 json 模块
  • 将Python对象编码为JSON字符串 json.dumps()：可以将Python的字典、列表等对象转换为JSON格式的字符串。
  • 将JSON字符串解码为Python对象 json.loads()：可以将JSON格式的字符串转换为Python的字典、列表等对象。
  • 将Python对象写入JSON文件 json.dump()
  • 从JSON文件读取Python对象 json.load()

# 小贴士

  • 在进行文件操作时，一定要注意文件的路径是否正确。
  • 不同的文件打开模式有不同的作用，要根据需求选择合适的模式（'r' 读，'w' 写，'a' 追加，'b' 二进制模式等）。
  • 处理JSON数据时，要确保JSON字符串的格式是正确的。

Python正则表达式入门指南：从小白到入门只需这一篇！🚀

哈喽各位小伙伴！今天我们要一起探索Python中一个非常强大且实用的工具——正则表达式！它就像一位超级侦探，能帮助我们在文本中快速准确地找到我们想要的信息。是不是听起来就很酷？😎 别担心，我会用最简单易懂的方式带你入门！

# 什么是正则表达式？🤔

简单来说，正则表达式（Regular Expression，常简写为 regex、regexp 或 RE）是一种用于匹配文本模式的特殊序列的字符。你可以把它想象成一种高级的搜索功能，不仅能找到固定的词语，还能根据你定义的“规则”找到符合特定模式的文本。

# Python中的 re 模块

要在Python中使用正则表达式，我们需要导入内置的 re 模块。就像你需要一把锤子才能钉钉子一样！🔨

# 核心概念：匹配模式与特殊字符

正则表达式的强大之处在于其灵活的匹配模式。我们可以使用一些特殊字符来定义这些模式：

  • 普通字符：大多数字母、数字、下划线等可以直接匹配自身。比如，正则表达式 "hello" 就能匹配文本中的 "hello"。
  • . (点号)：匹配任意单个字符（除了换行符 \n）。
  • ^ (脱字符)：匹配字符串的开头。例如，^hello 只会匹配以 "hello" 开头的字符串。
  • $ (美元符)：匹配字符串的结尾。例如，hello$ 只会匹配以 "hello" 结尾的字符串。
  • * (星号)：匹配前一个字符零次或多次。例如，ab*c 可以匹配 "ac"、"abc"、"abbc" 等。
  • + (加号)：匹配前一个字符一次或多次。例如，ab+c 可以匹配 "abc"、"abbc"，但不能匹配 "ac"。
  • ? (问号)：匹配前一个字符零次或一次。例如，ab?c 可以匹配 "ac" 或 "abc"。
  • {m}：匹配前一个字符恰好 m 次。例如，a{3} 只匹配 "aaa"。
  • {m, n}：匹配前一个字符至少 m 次，至多 n 次。
  • [] (字符集)：匹配方括号中的任意一个字符。例如，[abc] 可以匹配 "a"、"b" 或 "c"。
    • 在字符集中可以使用 - 表示范围，例如 [a-z] 匹配所有小写字母，[0-9] 匹配所有数字。
    • 在字符集开头使用 ^ 表示取反，例如 [^abc] 匹配除了 "a"、"b"、"c" 之外的任意字符。
  • \ (反斜杠)：用于转义特殊字符，使其失去特殊含义，或者引入特殊的预定义字符序列（见下文）。

# 常用的预定义字符序列

为了方便使用，re 模块提供了一些预定义的字符序列：

  • \d：匹配任意数字（等价于 [0-9]）。
  • \D：匹配任意非数字字符（等价于 [^0-9]）。
  • \s：匹配任意空白字符（包括空格、制表符 \t、换行符 \n 等）。
  • \S：匹配任意非空白字符。
  • \w：匹配任意字母、数字或下划线（等价于 [a-zA-Z0-9_]）。
  • \W：匹配任意非字母、数字或下划线。

# re 模块的常用函数

学会了匹配模式，我们来看看 re 模块中一些常用的函数：

  • re.search(pattern, string)：在字符串 string 中搜索第一个匹配 pattern 的位置，如果找到则返回一个匹配对象（Match object），否则返回 None。
  • re.match(pattern, string)：尝试从字符串 string 的开头匹配 pattern，如果匹配成功则返回一个匹配对象，否则返回 None。注意：它只从开头匹配！
  • re.findall(pattern, string)：在字符串 string 中找到所有匹配 pattern 的子串，并返回一个包含所有匹配项的列表。
  • re.sub(pattern, repl, string)：将字符串 string 中所有匹配 pattern 的子串替换为 repl。

# 实用技巧：原始字符串 (raw string)

在正则表达式中，反斜杠 \ 有特殊的含义。为了避免与Python字符串自身的转义冲突，我们通常在正则表达式前面加上 r 前缀，表示这是一个原始字符串。这样，反斜杠就不会被Python解释器进行额外的转义，而是直接传递给 re 模块。强烈建议养成使用原始字符串的习惯！👍

# 进阶：分组匹配

正则表达式还支持分组匹配，用小括号 () 包裹起来的部分会被认为是一个分组。我们可以使用匹配对象的 group() 方法来获取不同分组的内容。

# 注意事项

  • 正则表达式的语法比较灵活，需要多练习才能熟练掌握。
  • 对于复杂的匹配需求，可以逐步构建正则表达式。
  • 善用在线正则表达式测试工具，可以帮助你验证你的模式是否正确。

# 总结

正则表达式是处理文本数据的强大工具，掌握它能让你在文本搜索、数据提取、格式验证等方面事半功倍！希望今天的介绍能帮助你迈出学习Python正则表达式的第一步。快去动手尝试一下吧！加油！💪

# 认识你的函数朋友：不同类型的参数

在Python中，定义函数时，我们可以设置不同类型的参数，让函数能够接收各种各样的输入。主要有以下几种：

  • 位置传参（Positional Arguments）：这是最常见的参数类型啦！函数定义时的参数顺序很重要，你在调用函数时传入的值也会按照这个顺序一一对应。就像给你的朋友按名单顺序发礼物一样！🎁

  • 关键字传参（Keyword Arguments）：有时候我们记不清参数的顺序，或者想让代码更易读，就可以使用关键字传参。在调用函数时，直接指定参数名和对应的值。这样就不怕传错顺序啦！🔑

  • 默认值参数（Default Arguments）：在定义函数时，我们可以给一些参数设置默认值。如果在调用函数时没有给这些参数传值，它们就会使用默认值。就像你的朋友自带了某种小零食，如果你没带，就吃他的！🍬

  • 不定长位置参数（*args）：有时候我们不知道函数需要接收多少个位置参数，这时就可以使用*args。它会将传入的所有多余的位置参数打包成一个元组（tuple）。就像一个可以装很多东西的袋子！🎒

  • 不定长关键字参数（**kwargs）：类似地，**kwargs可以接收任意数量的关键字参数，并将它们打包成一个字典（dictionary）。每个传入的参数都需要指定参数名和值。就像一个可以存放各种标签和对应值的盒子！📦

# 划重点！参数使用的注意事项

  • 定义函数时，位置参数必须在默认值参数的前面。
  • 调用函数时，如果同时使用位置参数和关键字参数，位置参数必须在关键字参数的前面。
  • *args 和 **kwargs 通常放在参数列表的最后。

# 小彩蛋：匿名函数 lambda

除了用 def 定义函数，Python还提供了一种简洁的方式来创建匿名函数，也就是 lambda 函数。它通常用于一些简单的、只需要使用一次的函数。就像一个临时的便签！📝

循环中的“俄罗斯套娃”！嵌套循环！🎁🎁🎁

遍历小能手！for 循环！🚶‍♀️🚶🚶‍♂️

for 循环就像一个贴心的导游，可以带着我们一个一个地“参观”容器里的每个“景点”（元素）。容器可以是列表、字符串等等，只要里面有东西，for 循环就能把它遍历个遍！

for 循环还能指定循环次数哦！用 range() 函数就可以实现啦！就像我们跑步，可以指定跑几圈！🏃‍♀️

永不停歇的小马达！无限循环！<0xF0><0x9F><0xAA><0xB0>

关键时刻喊“停”！break！✋

“跳过”这个！continue！➡️

02-流程控制

Posted on 2025-04-26

兄弟们，今天我们来聊聊 Python 里的“流程控制”！🚦🚥 就像我们生活中的红绿灯，它能让我们的代码有条不紊地执行，该走哪条路，听谁的指令，都安排得明明白白！😎✨

让你的输出更精致！格式化输出小技巧！💅

有时候我们想让 print() 出来的文字更漂亮、更整齐，就需要用到格式化输出啦！就像我们化妆一样，精致一点才更吸睛！😉

  • 百分号 % 格式化： 这是 Python 比较“经典”的格式化方式啦！用不同的符号代表不同的数据类型：

    • %d：整数 (digit)
    • %f：浮点数 (float)
    • %s：字符串 (string)

  • F-string 格式化： 这是 Python 3.6 之后推出的超好用、超简洁的格式化方式！强烈推荐！👍 只需要在字符串前面加个 f，然后用大括号 {} 把变量包起来就好啦！简直不要太方便！💖

和你的程序“对话”！input() 输入函数！🗣️

我们的程序不能总是“自说自话”，有时候也需要我们用户输入一些信息，这时候就要用到 input() 函数啦！它就像一个“提问小助手”，等待我们输入内容。📝

敲黑板！重点来了！ input() 函数接收到的任何值都会变成字符串类型！即使你输入的是数字，它也会被当成文本来处理哦！所以，如果你想把输入的内容当成数字来用，就需要进行类型转换！

“七十二变”！数据类型转换！✨

有时候我们需要把数据从一种类型变成另一种类型，就像我们穿不同的衣服适应不同的场合一样！👗 Python 提供了一些内置函数来帮我们实现这个“变身”！

  • int(x)：把 x 转换成整数。注意！如果 x 是不能转换成整数的字符串（比如 “abc”），就会报错哦！⚠️
  • float(x)：把 x 转换成浮点数。
  • str(x)：把 x 转换成字符串。这个最常用啦，几乎所有类型都可以转换成字符串！

程序世界的“交通规则”！运算符！🚦

运算符就像我们生活中的各种符号，告诉程序该怎么“计算”或者“比较”不同的值。

  • 关系运算符： 用来比较两个值之间的关系，结果是布尔值（True 或 False）。就像我们比较谁高谁矮、谁大谁小一样！📏

    • > (大于)
    • < (小于)
    • >= (大于等于)
    • <= (小于等于)
    • == (等于)
    • != (不等于)

  • 逻辑运算符： 用来组合多个条件判断。就像我们说话时的“并且”、“或者”、“不是”！🔗

    • or (或者)：只要有一个条件为真，结果就为真。
    • and (并且)：所有条件都为真，结果才为真。
    • not (不是)：对一个条件取反，真变成假，假变成真。

  • 算术运算符： 用来进行数学计算。就像我们平时做的加减乘除！➕➖✖️➗

    • + (加)
    • - (减)
    • * (乘)
    • / (除)
    • // (整除，结果只保留整数部分)
    • % (取余数，就是除法运算后的余数)
    • ** (幂运算，就是几的几次方)

  • 赋值运算符： 用来给变量赋值。最常见的就是 = 等号啦！当然还有一些更简洁的写法：

    • = (等于)
    • += (加等于，相当于 a = a + b)
    • -= (减等于，相当于 a = a - b)
    • *= (乘等于，相当于 a = a * b)
    • /= (除等于，相当于 a = a / b)
    • //= (整除等于)
    • %= (取余等于)
    • **= (幂等于)

让你的代码“聪明”起来！if 语句！🧠

if 语句就像我们生活中的“如果…就…”的判断。它能让我们的代码根据不同的条件执行不同的操作，变得更智能！

  • if 单分支结构： 如果条件成立，就执行一些代码。

  • if 双分支结构： 如果条件成立，执行一些代码；否则，执行另一些代码。就像我们选择题，要么 A，要么 B！

  • if 多分支结构： 有多个条件需要判断的时候用！就像我们根据不同的分数评判等级一样！

让你的代码“循环”起来！while 语句！🔄

while 循环就像我们生活中的“重复…直到…”的过程。只要条件成立，它就会一直执行一些代码，直到条件不成立才停止。
