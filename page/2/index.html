  • еҹәзЎҖеҠһе…¬иҪҜд»¶е®үиЈ…пјҡ иҝҷдёӘеҳӣпјҢзӣёдҝЎе§җеҰ№д»¬йғҪжҳҜ е°ҸиғҪжүӢе•ҰпјҒжҸҗй«ҳж•ҲзҺҮзҡ„е·Ҙе…·дҪ“зҺ°дәҶеҜ№ж—¶й—ҙзҡ„й«ҳж•ҲеҲ©з”Ёрҹ’»

Python зҡ„еүҚдё–д»Ҡз”ҹ & е®ғзҡ„вҖңи¶…иғҪеҠӣвҖқпјҒрҹҰёвҖҚвҷҖпёҸ

иҜқиҜҙ Python иҝҷй—ЁиҜӯиЁҖпјҢе°ұеғҸдёҖдёӘжё©жҹ”еҸҲејәеӨ§зҡ„е°Ҹе§җе§җпјҒеҘ№жңүеҫҲеӨҡдјҳзӮ№пјҢи®©дәәдёҖз”Ёе°ұзҲұдёҠпјҒрҹҳҚ

Python зҡ„дјҳзӮ№ вңЁпјҡ

  1. з®ҖеҚ•жҳ“еӯҰпјҡ иҜӯжі•и¶…зә§з®ҖжҙҒжё…жҷ°пјҢе°ұеғҸе’Ңе°Ҹе§җеҰ№иҒҠеӨ©дёҖж ·иҮӘз„¶пјҒеҶҚд№ҹдёҚз”ЁеҜ№зқҖйӮЈдәӣеӨ©д№ҰиҲ¬зҡ„д»Јз ҒеҸ‘ж„Ғе•ҰпјҒрҹ“–вһЎпёҸрҹҳҠ
  2. еҸҜз§»жӨҚжҖ§ејәпјҡ е°ұеғҸдёҖдёӘзҷҫжҗӯеҚ•е“ҒпјҢPython д»Јз ҒеҮ д№ҺеҸҜд»ҘеңЁеҗ„з§Қж“ҚдҪңзі»з»ҹдёҠе®ҢзҫҺиҝҗиЎҢпјҒдёҖж¬Ўзј–еҶҷпјҢеҲ°еӨ„йғҪиғҪз”ЁпјҒрҹҢҚ
  3. еә“и¶…дё°еҜҢпјҡ Python жӢҘжңүдёҖдёӘ аҰ¬аҰҝаҰ¶аҰҫаҰІ (е·ЁеӨ§зҡ„ж„ҸжҖқпјҒ) зҡ„вҖңе·Ҙе…·з®ұвҖқпјҢеҗ„з§Қеҗ„ж ·зҡ„еә“еә”жңүе°ҪжңүпјҢж— и®әжҳҜжғіжҗһдёӘзҪ‘з«ҷгҖҒеҲҶжһҗж•°жҚ®гҖҒиҝҳжҳҜзҺ©зҺ©дәәе·ҘжҷәиғҪпјҢйғҪиғҪжүҫеҲ°еҗҲйҖӮзҡ„вҖңе·Ҙе…·вҖқпјҒрҹ§°
  4. еә”з”ЁеңәжҷҜе№ҝжіӣпјҡ з®ҖзӣҙжҳҜе…ЁиғҪ ACEпјҒWeb еә”з”ЁејҖеҸ‘гҖҒеӨ§ж•°жҚ®ејҖеҸ‘гҖҒж•°жҚ®еҲҶжһҗгҖҒдәәе·ҘжҷәиғҪгҖҒжЎҢйқўиҪҜд»¶гҖҒиҝҗз»ҙжөӢиҜ•вҖҰ е“ӘйҮҢйғҪиғҪзңӢеҲ°еҘ№зҡ„иә«еҪұпјҒз®ҖзӣҙжҳҜзЁӢеәҸеӘӣзҡ„зҷҫеҸҳжҲҳиўҚпјҒрҹ‘—

Python зҡ„е°ҸзјәзӮ№ рҹӨ«пјҡ

  • жү§иЎҢж•ҲзҺҮзЁҚж…ўпјҡ и§ЈйҮҠеһӢиҜӯиЁҖеҳӣпјҢе°ұеғҸжҲ‘д»¬иҜҙиҜқйңҖиҰҒзҝ»иҜ‘дёҖж ·пјҢдјҡзЁҚеҫ®ж…ўдёҖзӮ№зӮ№гҖӮдҪҶеҜ№дәҺжҲ‘д»¬ж—Ҙеёёеә”з”ЁжқҘиҜҙпјҢеҮ д№Һж„ҹи§үдёҚеҲ°е•ҰпјҒрҹҗҢвһЎпёҸрҹҸғвҖҚвҷҖпёҸ

з»ҷдҪ зҡ„д»Јз ҒеҠ зӮ№вҖңжӮ„жӮ„иҜқвҖқпјҒрҹ’¬

еҶҷд»Јз Ғзҡ„ж—¶еҖҷпјҢдёәдәҶж–№дҫҝиҮӘе·ұе’ҢеҲ«дәәзҗҶи§ЈпјҢжҲ‘д»¬з»ҸеёёдјҡеҠ дёҖдәӣвҖңжіЁйҮҠвҖқпјҢе°ұеғҸеңЁиҒҠеӨ©и®°еҪ•йҮҢеҠ дёӘиЎЁжғ…жҲ–иҖ…еӨҮжіЁдёҖж ·пјҒрҹҳү

  • еҚ•иЎҢжіЁйҮҠпјҡ з”Ё # ејҖеӨҙпјҢиҝҷдёҖиЎҢеҗҺйқўзҡ„еҶ…е®№ Python е°ұзӣҙжҺҘеҝҪз•Ҙе•ҰпјҒе°ұеғҸжҲ‘д»¬зҡ„е°Ҹз§ҳеҜҶпјҢеҸӘжңүиҮӘе·ұзҹҘйҒ“пјҒрҹӨ«
  • еӨҡиЎҢжіЁйҮҠпјҡ з”ЁдёүдёӘеҸҢеј•еҸ· """ еҢ…иЈ№иө·жқҘпјҢеҸҜд»ҘеҶҷеҫҲеӨҡеҫҲеӨҡиЎҢжіЁйҮҠпјҒе°ұеғҸжҲ‘д»¬зҡ„е°Ҹж—Ҙи®°жң¬пјҒрҹ“–

з»ҷдҪ зҡ„ж•°жҚ®жүҫдёӘвҖңе®¶вҖқпјҒеҸҳйҮҸе’Ңзұ»еһӢпјҒрҹҸ рҹҸ·пёҸ

еңЁ Python йҮҢпјҢжҲ‘д»¬иҰҒеӯҳеӮЁеҗ„з§Қеҗ„ж ·зҡ„ж•°жҚ®пјҢе°ұйңҖиҰҒз”ЁеҲ°вҖңеҸҳйҮҸвҖқиҝҷдёӘе°Ҹе®№еҷЁпјҒе°ұеғҸжҲ‘д»¬зҡ„еҢ–еҰҶеҢ…пјҢеҸҜд»ҘиЈ…еҗ„з§ҚдёҚеҗҢзҡ„еҢ–еҰҶе“ҒпјҒрҹ’„ иҖҢдё”иҝҷдёӘвҖңеҢ–еҰҶеҢ…вҖқйҮҢзҡ„дёңиҘҝжҳҜеҸҜд»ҘйҡҸж—¶жӣҙжҚўзҡ„е“ҰпјҒрҹҳү

  • еҸҳйҮҸпјҡ з”ЁжқҘеӯҳеӮЁж•°жҚ®зҡ„вҖңе°Ҹзӣ’еӯҗвҖқпјҢйҮҢйқўзҡ„еҶ…е®№еҸҜд»ҘйҡҸж—¶ж”№еҸҳпјҢдҪҶдёҖж¬ЎеҸӘиғҪиЈ…дёҖдёӘвҖңдёңиҘҝвҖқе“ҰпјҒрҹ“Ұ
  • зұ»еһӢпјҡ дёҚеҗҢзҡ„ж•°жҚ®жңүдёҚеҗҢзҡ„вҖңзұ»еһӢвҖқпјҢе°ұеғҸдёҚеҗҢзҡ„еҢ–еҰҶе“ҒжңүдёҚеҗҢзҡ„еҠҹж•ҲдёҖж ·пјҒрҹ§ҙ
    • intпјҡж•ҙж•°пјҢе°ұжҳҜжІЎжңүе°Ҹж•°зӮ№зҡ„ж•°еӯ—е•ҰпјҒжҜ”еҰӮпјҡ1, 100, -5гҖӮ
    • floatпјҡжө®зӮ№ж•°пјҢе°ұжҳҜеёҰе°Ҹж•°зӮ№зҡ„ж•°еӯ—пјҒжҜ”еҰӮпјҡ3.14, 1.68гҖӮ
    • boolпјҡеёғе°”еҖјпјҢеҸӘжңүдёӨдёӘеҖјпјҡTrueпјҲзңҹпјүе’Ң FalseпјҲеҒҮпјүпјҢе°ұеғҸжҲ‘д»¬д»ҠеӨ©зҡ„еҝғжғ…пјҒрҹҳҠ/рҹҳ”
    • strпјҡеӯ—з¬ҰдёІпјҢе°ұжҳҜдёҖдёІж–Үжң¬пјҢз”Ёеј•еҸ·пјҲеҚ•еј•еҸ·жҲ–еҸҢеј•еҸ·пјүеҢ…иө·жқҘпјҒжҜ”еҰӮпјҡвҖқPythonвҖқ, вҖҳзңҹйҰҷвҖҷгҖӮ

з»ҷдҪ зҡ„вҖңе®қиҙқвҖқиө·дёӘеҘҪеҗ¬зҡ„еҗҚеӯ—пјҒж ҮиҜҶз¬Ұе’Ңе‘ҪеҗҚи§„иҢғпјҒрҹҸ·пёҸ

жҲ‘д»¬еҲӣе»әзҡ„еҸҳйҮҸгҖҒеҮҪж•°зӯүзӯүпјҢйғҪйңҖиҰҒиө·дёӘеҗҚеӯ—пјҢиҝҷе°ұжҳҜвҖңж ҮиҜҶз¬ҰвҖқпјҒиө·дёӘеҘҪеҗҚеӯ—е°ұеғҸз»ҷжҲ‘д»¬зҡ„е°Ҹе® зү©иө·дёӘеҸҜзҲұзҡ„жҳөз§°дёҖж ·пјҢж–№дҫҝжҲ‘д»¬и®°дҪҸе’ҢдҪҝз”ЁпјҒрҹҗ¶

  1. и§ҒеҗҚзҹҘж„ҸпјҒ иө·зҡ„еҗҚеӯ—иҰҒиғҪдёҖзңјзңӢеҮәжҳҜе№Іе•Ҙз”Ёзҡ„пјҒжҜ”еҰӮеӯҳеҗҚеӯ—зҡ„еҸҳйҮҸе°ұеҸ« nameпјҢеӯҳе№ҙйҫ„зҡ„е°ұеҸ« ageпјҢиҝҷж ·д»Јз ҒжүҚжӣҙе®№жҳ“иҜ»жҮӮпјҒе°ұеғҸжҲ‘д»¬з»ҷж–Үд»¶е‘ҪеҗҚдёҖж ·пјҢиҰҒжё…жҷ°жҳҺдәҶпјҒрҹ“Ӯ
  2. еӯ—жҜҚгҖҒж•°еӯ—гҖҒдёӢеҲ’зәҝз»„жҲҗпјҡ жҲ‘д»¬зҡ„вҖңжҳөз§°вҖқеҸӘиғҪеҢ…еҗ«иҝҷдёүз§Қе“ҰпјҒдёҚиғҪжңүз©әж јжҲ–иҖ…е…¶д»–зү№ж®Ҡз¬ҰеҸ·пјҒрҹҡ«
  3. ж•°еӯ—дёҚиғҪејҖеӨҙпјҒ вҖңжҳөз§°вҖқдёҚиғҪд»Ҙж•°еӯ—ејҖеӨҙе“ҰпјҒрҹҷ…вҖҚвҷҖпёҸ
  4. еҢәеҲҶеӨ§е°ҸеҶҷпјҒ myName е’Ң MyName жҳҜдёӨдёӘдёҚеҗҢзҡ„вҖңжҳөз§°вҖқе“ҰпјҒрҹ‘Җ

  • жңҖеҗҺиҫҹдёӘи°ЈпјҒйҳҝйҮҢдә‘е’Ңж·ҳе®қиҷҪз„¶йғҪжҳҜйҳҝйҮҢйӣҶеӣўзҡ„пјҢдҪҶеҸҜжҳҜзӢ¬з«ӢиҝҗиҗҘзҡ„йғЁй—Ёе“ҰпјҒеҲ«жҗһж··е•ҰпјҒрҹҷ…вҖҚвҷҖпёҸ

з®ҖеҚ•жқҘиҜҙпјҢдә‘жңҚеҠЎеҷЁе°ұеғҸдҪ еңЁдә‘з«Ҝз§ҹдәҶдёҖеҸ°и¶…еҺүе®ізҡ„вҖңиҷҡжӢҹз”өи„‘вҖқпјҒрҹ’» е®ғеҹәдәҺдә‘и®Ўз®—жҠҖжңҜпјҢеҸҜд»Ҙеј№жҖ§ең°и°ғж•ҙй…ҚзҪ®пјҢжғіеӨ§е°ұеӨ§пјҢжғіе°Ҹе°ұе°ҸпјҢи¶…зҒөжҙ»пјҒе°ұеғҸжҲ‘д»¬еҘіз”ҹзҡ„иЎЈж©ұпјҢжғіеЎһеӨҡе°‘е°ұеӨҡе°‘пјҒпјҲеҘҪеғҸе“ӘйҮҢдёҚеҜ№пјҹрҹҳӮпјү

еёёи§Ғдә‘жңҚеҠЎеҷЁвҖңжҲҝдёңвҖқжңүе“ӘдәӣпјҹрҹҸўрҹҢҚ

жғіеңЁдә‘з«Ҝе®үе®¶пјҹйҖүжӢ©еҸҜдёҚе°‘е‘ўпјҒ

  • еӣҪеҶ…зҹҘеҗҚвҖңжҲҝдёңвҖқпјҡ йҳҝйҮҢдә‘гҖҒи…ҫи®Ҝдә‘гҖҒеҚҺдёәдә‘гҖҒUCloudгҖҒзҷҫеәҰдә‘гҖҒйҮ‘еұұдә‘гҖҒдә¬дёңдә‘пјҢдёӘдёӘе®һеҠӣйғҪдёҚе®№е°Ҹи§‘пјҒрҹҸҳпёҸ
  • еӣҪйҷ…еӨ§зүҢвҖңжҲҝдёңвҖқпјҡ AWS (дәҡй©¬йҖҠдә‘)гҖҒеҫ®иҪҜ AzureгҖҒGCP (и°·жӯҢдә‘)пјҢжҠҖжңҜйӮЈжҳҜжқ жқ зҡ„пјҒрҹҢҚ

еҒ·еҒ·е‘ҠиҜүдҪ пјҢеңЁе…ЁзҗғеёӮеңәпјҢAWS жҳҜиҖҒеӨ§е“ҰпјҒиҖҢеңЁжҲ‘д»¬еӣҪеҶ…пјҢйҳҝйҮҢдә‘еҸҜжҳҜзЁіеқҗз¬¬дёҖжҠҠдәӨжӨ…пјҒеҺүе®ідәҶжҲ‘зҡ„дә‘пјҒрҹ‘Ҹ

дёәд»Җд№ҲеӨ§е®¶йғҪзҲұдә‘жңҚеҠЎеҷЁпјҹе®ғзҡ„вҖңи¶…иғҪеҠӣвҖқеӨ§жҸӯз§ҳпјҒрҹҰёвҖҚвҷҖпёҸ

дә‘жңҚеҠЎеҷЁд№ӢжүҖд»Ҙиҝҷд№ҲеҸ—ж¬ўиҝҺпјҢеҪ“з„¶жҳҜеӣ дёәе®ғжңүеҫҲеӨҡиҝҮдәәд№ӢеӨ„е•ҰпјҒ

  1. зңҒй’ұе°ҸиғҪжүӢпјҒрҹ’°пјҡ жҢүйңҖд»ҳиҙ№пјҢз”ЁеӨҡе°‘иҠұеӨҡе°‘пјҢеҶҚд№ҹдёҚз”ЁжӢ…еҝғд№°дёҖе Ҷи®ҫеӨҮеҗғзҒ°е•ҰпјҒз®ҖзӣҙжҳҜзІҫжү“з»Ҷз®—е§җеҰ№зҡ„зҰҸйҹіпјҒрҹ’–
  2. зҒөжҙ»е°ҸеӨ©еҗҺпјҒрҹ’ғпјҡ дёҡеҠЎйҮҸеӨ§дәҶпјҹдёҖй”®еҚҮзә§й…ҚзҪ®пјҒжҗһжҙ»еҠЁйңҖиҰҒдёҙж—¶жү©е®№пјҹжІЎй—®йўҳпјҒе°ұеғҸжҲ‘д»¬з©ҝжҗӯдёҖж ·пјҢйҡҸж—¶еҲҮжҚўйЈҺж јпјҒрҹ‘—
  3. е®үе…Ёж„ҹзҲҶжЈҡпјҒрҹӣЎпёҸпјҡ дё“дёҡзҡ„е®үе…ЁжҺӘж–Ҫе’Ңж•°жҚ®еӨҮд»ҪжңәеҲ¶пјҢе…Ёж–№дҪҚдҝқжҠӨдҪ зҡ„ж•°жҚ®пјҒеҶҚд№ҹдёҚз”ЁжӢ…еҝғз”өи„‘еқҸдәҶиө„ж–ҷе…ЁжІЎдәҶпјҒрҹ’Ә
  4. жҮ’дәәеҝ…еӨҮпјҒрҹӣӢпёҸпјҡ еҗ„з§ҚиҮӘеҠЁеҢ–з®ЎзҗҶе·Ҙе…·пјҢиҝҗз»ҙеҸҳеҫ—и¶…з®ҖеҚ•пјҒдёҚз”ЁжҳҜ IT еӨ§дҪ¬д№ҹиғҪиҪ»жқҫз®ЎзҗҶпјҒвңҢпёҸ
  5. еҲӣж–°еҠ йҖҹеҷЁпјҒрҹҡҖпјҡ зҒөжҙ»зҡ„и®Ўз®—иө„жәҗпјҢи®©дҪ еҸҜд»Ҙдё“жіЁдәҺжҗһдәӢжғ…пјҢеӨ§иғҶеҲӣж–°пјҒрҹ’Ў

жҢ‘йҖүдә‘жңҚеҠЎеҷЁпјҢиҝҷдәӣвҖңдҪ“жЈҖжҠҘе‘ҠвҖқиҰҒзңӢд»”з»ҶпјҒрҹ§җ

йҖүдә‘жңҚеҠЎеҷЁе°ұеғҸйҖүеҜ№иұЎпјҢиҰҒзңӢеҗҲдёҚеҗҲйҖӮпјҒрҹҳҺ иҝҷдәӣвҖңдҪ“жЈҖжҠҘе‘ҠвҖқеҸҜиҰҒи®Өзңҹз ”з©¶пјҡ

  1. вҖңеӨ§и„‘вҖқиҰҒиҒӘжҳҺпјҒрҹ§  (и®Ўз®—иғҪеҠӣ)
    • CPU (ж ёж•°)пјҡ еҰӮжһңдҪ зҡ„еә”з”ЁйңҖиҰҒеӨ„зҗҶеӨ§йҮҸи®Ўз®—пјҢжҜ”еҰӮеҲҶжһҗж•°жҚ®гҖҒи·‘ AIпјҢйӮЈе°ұйҖү CPU ж ёеҝғеӨҡзҡ„пјҒрҹ’Ә
    • еҶ…еӯҳ (еӨ§е°Ҹ)пјҡ еә”з”ЁиҰҒеӨ„зҗҶеҫҲеӨҡж•°жҚ®жҲ–иҖ…еҗҢж—¶иҝҗиЎҢеҫҲеӨҡзЁӢеәҸпјҹйӮЈе°ұйҖүеҶ…еӯҳеӨ§зҡ„пјҒрҹ§ 
  2. вҖңд»“еә“вҖқиҰҒеӨҹеӨ§пјҒрҹ“Ұ (еӯҳеӮЁе®№йҮҸ)
    • зЈҒзӣҳзұ»еһӢпјҡ йҳҝйҮҢдә‘жҸҗдҫӣдәҶ SSDгҖҒй«ҳж•Ҳдә‘зӣҳе’Ңжҷ®йҖҡдә‘зӣҳгҖӮSSD йҖҹеәҰжңҖеҝ«пјҢйҖӮеҗҲеҜ№йҖҹеәҰиҰҒжұӮй«ҳзҡ„еңәжҷҜпјӣй«ҳж•Ҳдә‘зӣҳжҖ§д»·жҜ”й«ҳпјӣжҷ®йҖҡдә‘зӣҳе°ұйҖӮеҗҲеҜ№йҖҹеәҰиҰҒжұӮдёҚй«ҳзҡ„еңәжҷҜе•ҰпјҒж №жҚ®дҪ зҡ„йңҖжұӮжқҘйҖүе“ҰпјҒрҹҳү
    • еӯҳеӮЁе®№йҮҸпјҡ иҰҒеӯҳеҫҲеӨҡж–Үд»¶жҲ–иҖ…з»ҸеёёиҜ»еҶҷж•°жҚ®пјҹйӮЈе°ұйҖүе®№йҮҸеӨ§зҡ„пјҒрҹ’ҫ
  3. вҖңзҪ‘йҖҹвҖқиҰҒз»ҷеҠӣпјҒрҹ“¶ (зҪ‘з»ңжҖ§иғҪ)
    • еёҰе®Ҫпјҡ еҰӮжһңдҪ зҡ„еә”з”ЁйңҖиҰҒеӨ„зҗҶеҫҲеӨҡзҪ‘з»ңиҜ·жұӮжҲ–иҖ…дј иҫ“еӨ§йҮҸж•°жҚ®пјҢйӮЈе°ұйҖүеёҰе®Ҫй«ҳзҡ„пјҒрҹҡҖ

дә‘жңҚеҠЎеҷЁжҖҺд№Ҳз®—й’ұпјҹрҹ’°рҹ’і

дә‘жңҚеҠЎеҷЁзҡ„ж”¶иҙ№ж–№ејҸд№ҹеҫҲзҒөжҙ»е“ҰпјҒ

  • жҢүйҮҸд»ҳиҙ№пјҡ з”ЁеӨҡе°‘д»ҳеӨҡе°‘пјҢе°ұеғҸжү“иҪҰдёҖж ·пјҒрҹҡ• йҖӮеҗҲжңүзӘҒеҸ‘дёҡеҠЎйҮҸжҲ–иҖ…дёҙж—¶жөӢиҜ•зҡ„еңәжҷҜпјҒ
  • еҢ…е№ҙеҢ…жңҲпјҡ жҸҗеүҚйў„д»ҳдёҖж®өж—¶й—ҙзҡ„иҙ№з”ЁпјҢе°ұеғҸеҢ…жңҲдјҡе‘ҳеҚЎпјҒрҹ’і йҖӮеҗҲй•ҝжңҹзЁіе®ҡиҝҗиЎҢзҡ„жңҚеҠЎпјҢжҜ”еҰӮдҪ зҡ„е°ҸзҪ‘з«ҷпјҒ
  • жҠўеҚ ејҸе®һдҫӢпјҡ жҚЎжјҸе°ҸиғҪжүӢпјҒрҹҳҺ йҳҝйҮҢдә‘дјҡеҮәе”®дёҖдәӣз©әй—Ізҡ„и®Ўз®—иө„жәҗпјҢд»·ж ји¶…еҲ’з®—пјҒдҪҶжҳҜеҸҜиғҪдјҡиў«йҡҸж—¶еӣһж”¶пјҢйҖӮеҗҲеҜ№зЁіе®ҡжҖ§иҰҒжұӮдёҚй«ҳзҡ„еңәжҷҜпјҒ

е®һжҲҳжј”з»ғпјҒжүӢжҠҠжүӢж•ҷдҪ вҖңдә‘е…»еЁғвҖқпјҒрҹ‘¶вһЎпёҸрҹ’Ә

иҜҙдәҶиҝҷд№ҲеӨҡзҗҶи®әпјҢдёҚеҰӮжқҘзӮ№е®һйҷ…зҡ„пјҒжҲ‘д»¬жқҘдёҖиө·вҖңдә‘е…»вҖқдёӨеҸ°е°ҸжңҚеҠЎеҷЁпјҒ

  1. з”ійўҶдә‘жңҚеҠЎеҷЁпјҡ еңЁйҳҝйҮҢдә‘дёҠз”іиҜ·дёӨеҸ°дә‘жңҚеҠЎеҷЁпјҒдёҖеҸ°з”ЁжқҘйғЁзҪІ NginxпјҢеҒҡжҲ‘д»¬зҪ‘з«ҷзҡ„вҖңй—ЁйқўвҖқпјҲеҸҚеҗ‘д»ЈзҗҶпјүпјӣеҸҰдёҖеҸ°з”ЁжқҘи·‘жҲ‘д»¬зҡ„еә”з”ЁпјҢиҝҳиҰҒиЈ…дёӘе°Ҹж•°жҚ®еә“пјҒ
  2. й…ҚзҪ®е®үе…Ёз»„пјҡ е®үе…Ёз»„е°ұеғҸжҲ‘д»¬з”өи„‘зҡ„йҳІзҒ«еўҷпјҒрҹӣЎпёҸ и®ҫзҪ®еҘҪи§„еҲҷпјҢжүҚиғҪдҝқжҠӨжҲ‘д»¬зҡ„жңҚеҠЎеҷЁдёҚиў«еқҸдәәе…ҘдҫөпјҒ
  3. й…ҚзҪ®дё»жңәеҗҚе’ҢеҶ…зҪ‘еҹҹеҗҚи§Јжһҗпјҡ з»ҷжңҚеҠЎеҷЁиө·дёӘеҘҪеҗ¬зҡ„еҗҚеӯ—пјҢеҶҚи®ҫзҪ®дёҖдёӢеҶ…зҪ‘зҡ„вҖңжҳөз§°вҖқпјҢж–№дҫҝе®ғд»¬дә’зӣёи®ӨиҜҶпјҒрҹӨқ
  4. иҝһжҺҘжңҚеҠЎеҷЁпјҡ дёӢиҪҪйҳҝйҮҢдә‘зҡ„з»Ҳз«Ҝе·Ҙе…·пјҢжҲ–иҖ…з”Ёе…¶д»–зҡ„з»Ҳз«ҜиҪҜд»¶пјҢиҝһжҺҘеҲ°жҲ‘д»¬зҡ„дә‘жңҚеҠЎеҷЁпјҒе°ұеғҸиҝңзЁӢйҒҘжҺ§дҪ зҡ„з”өи„‘дёҖж ·пјҒрҹ•№пёҸ
  5. е®үиЈ…иҪҜд»¶пјҡ еңЁжңҚеҠЎеҷЁдёҠе®үиЈ…жҲ‘д»¬йңҖиҰҒзҡ„ NginxгҖҒж•°жҚ®еә“зӯүзӯүиҪҜд»¶пјҒе°ұеғҸз»ҷдҪ зҡ„вҖңдә‘з”өи„‘вҖқиЈ…дёҠеҗ„з§Қ AppпјҒ
  6. й…ҚзҪ®дә‘зӣ‘жҺ§е’ҢжҠҘиӯҰпјҡ з»ҷжҲ‘д»¬зҡ„вҖңдә‘з”өи„‘вҖқиЈ…дёҠвҖңзңјзқӣвҖқе’ҢвҖңиҖіжңөвҖқпјҒрҹ‘Җрҹ‘Ӯ зӣ‘жҺ§е®ғзҡ„иҝҗиЎҢзҠ¶жҖҒпјҢдёҖж—ҰеҮәй—®йўҳе°ұеҸҠж—¶еҸ‘еҮәиӯҰжҠҘпјҒиҝҳеҸҜд»Ҙй…ҚзҪ®и¶…й…·зӮ«зҡ„еҸҜи§ҶеҢ–еӨ§еұҸпјҢе®һж—¶жҹҘзңӢжңҚеҠЎеҷЁзҡ„еҗ„йЎ№жҢҮж ҮпјҒрҹ“Ҡ
  7. йӘҢиҜҒе’Ңе…ій—ӯжңҚеҠЎеҷЁпјҡ зЎ®и®ӨдёҖеҲҮжӯЈеёёеҗҺпјҢе°ұеҸҜд»ҘжҡӮж—¶е…ій—ӯжңҚеҠЎеҷЁе•ҰпјҒе°ұеғҸз”Ёе®Ңз”өи„‘е…іжңәдёҖж ·пјҒрҹ‘Ӣ

е…„ејҹд»¬пјҢз»ҸиҝҮиҝҷ 8 еӨ©зҡ„иҜҫзЁӢи®ҫи®ЎпјҢжҲ‘д»¬е·Із»ҸжҲҗеҠҹе…Ҙй—Ёиҝҗз»ҙзҡ„еҹәзЎҖе•ҰпјҒрҹҺү жҳҜдёҚжҳҜеҫҲзҘһеҘҮе‘ўпјҹжңҹеҫ…жҲ‘д»¬дёӢж¬ЎеҶҚи§ҒпјҒрҹ’– еңЁжңӘжқҘзҡ„ж—ҘеӯҗйҮҢпјҢжҲ‘дјҡз»§з»ӯеҲҶдә«жҲ‘зҡ„иҝҗз»ҙз»ҸйӘҢе’ҢжҖ»з»“пјҢе’ҢеӨ§е®¶дёҖиө·еңЁжҠҖжңҜзҡ„дё–з•ҢйҮҢй—Әй—ӘеҸ‘е…үпјҒвңЁ

07-зі»з»ҹжңҚеҠЎз®ЎзҗҶ

Posted on 2025-04-26

е…„ејҹд»¬пјҢд»ҠеӨ©жқҘиҒҠиҒҠ Linux зі»з»ҹйҮҢйӮЈдәӣи®©з”өи„‘вҖңд№–д№–еҗ¬иҜқвҖқзҡ„е°Ҹз§ҳеҜҶпјҒрҹӨ«вңЁ жҺҢжҸЎдәҶиҝҷдәӣпјҢдҪ д№ҹиғҪиҪ»жқҫе®үжҺ’з”өи„‘зҡ„ж—Ҙеёёе·ҘдҪңе•ҰпјҒрҹ’»рҹ’–

и®©жңҚеҠЎи·‘иө·жқҘпјҒsystemctl еӨ§жі•пјҒрҹҡҖ

еңЁ Linux иҝҷдёӘзҘһеҘҮзҡ„дё–з•ҢйҮҢпјҢеҗ„з§ҚеҠҹиғҪйғҪеғҸдёҖдёӘдёӘе°Ҹе°Ҹзҡ„вҖңжңҚеҠЎвҖқеңЁеҗҺеҸ°й»ҳй»ҳиҝҗиЎҢгҖӮжҜ”еҰӮдҪ жғіиҝңзЁӢиҝһжҺҘдҪ зҡ„з”өи„‘пјҹйӮЈе°ұйңҖиҰҒ sshd иҝҷдёӘвҖңе°ҸеҠ©жүӢвҖқдёҖзӣҙеңЁеҗҺеҸ°еҫ…е‘ҪпјҒжҖҺд№Ҳи®©е®ғд»¬еҗҜеҠЁгҖҒеҒңжӯўгҖҒйҮҚеҗҜпјҢз”ҡиҮіжҹҘзңӢзҠ¶жҖҒе‘ўпјҹе°ұйқ жҲ‘д»¬ејәеӨ§зҡ„ systemctl е‘Ҫд»Өе•ҰпјҒрҹ’Ә

е°ұеғҸйҒҘжҺ§еҷЁдёҖж ·пјҢsystemctl еҸҜд»ҘжҺ§еҲ¶иҝҷдәӣвҖңжңҚеҠЎвҖқзҡ„зҠ¶жҖҒпјҡ

  • еҗҜеҠЁжңҚеҠЎпјҡ sudo systemctl start <жңҚеҠЎеҗҚ> жҜ”еҰӮеҗҜеҠЁжҲ‘д»¬зҡ„иҝңзЁӢиҝһжҺҘе°ҸеҠ©жүӢпјҡsudo systemctl start sshd еҗҜеҠЁеҗҺдҪ зҡ„з”өи„‘е°ұиғҪжҺҘеҸ—иҝңзЁӢиҝһжҺҘе•ҰпјҒрҹ”—
  • еҒңжӯўжңҚеҠЎпјҡ sudo systemctl stop <жңҚеҠЎеҗҚ> жҡӮж—¶дёҚйңҖиҰҒжҹҗдёӘжңҚеҠЎдәҶпјҹе°ұи®©е®ғе…Ҳдј‘жҒҜдёҖдёӢпјҡsudo systemctl stop sshd
  • йҮҚеҗҜжңҚеҠЎпјҡ sudo systemctl restart <жңҚеҠЎеҗҚ> жңүж—¶еҖҷжңҚеҠЎеҮәзӮ№е°ҸзҠ¶еҶөпјҢйҮҚеҗҜдёҖдёӢе°ұеҘҪе•ҰпјҒе°ұеғҸз»ҷ App вҖңеҲ·ж–°вҖқдёҖдёӢпјҡsudo systemctl restart sshd
  • жҹҘзңӢжңҚеҠЎзҠ¶жҖҒпјҡ sudo systemctl status <жңҚеҠЎеҗҚ> жғізҹҘйҒ“жҹҗдёӘжңҚеҠЎзҺ°еңЁжӯЈдёҚжӯЈеёёе·ҘдҪңпјҹзңӢзңӢе®ғзҡ„зҠ¶жҖҒе°ұзҹҘйҒ“е•ҰпјҒдјҡжҳҫзӨәиҝҗиЎҢзҠ¶жҖҒгҖҒиҝӣзЁӢдҝЎжҒҜзӯүзӯүпјҢи¶…иҜҰз»ҶпјҒрҹ”Қ

е®ҡж—¶д»»еҠЎпјҹдәӨз»ҷ crontab еҮҶжІЎй”ҷпјҒвҸ°

жңүж—¶еҖҷжҲ‘д»¬еёҢжңӣз”өи„‘еңЁзү№е®ҡзҡ„ж—¶й—ҙиҮӘеҠЁеё®жҲ‘д»¬еҒҡдёҖдәӣдәӢжғ…пјҢжҜ”еҰӮжҜҸеӨ©еҮҢжҷЁеӨҮд»Ҫж•°жҚ®гҖҒе®ҡж—¶жё…зҗҶдёҙж—¶ж–Үд»¶зӯүзӯүгҖӮиҝҷж—¶еҖҷе°ұиҰҒиҜ·еҮәжҲ‘д»¬зҡ„вҖңе®ҡж—¶д»»еҠЎе°Ҹз®Ўе®¶вҖқвҖ”вҖ” crontab е•ҰпјҒрҹ•°пёҸ

дҪ еҸҜд»Ҙзј–иҫ‘дёҖдёӘеҸ«еҒҡ вҖңcrontabвҖқ зҡ„ж–Үд»¶пјҢе‘ҠиҜүз”өи„‘еңЁд»Җд№Ҳж—¶й—ҙжү§иЎҢд»Җд№Ҳе‘Ҫд»ӨгҖӮ

зј–иҫ‘дҪ зҡ„е®ҡж—¶д»»еҠЎеҲ—иЎЁпјҡ

з¬¬дёҖж¬ЎиҝҗиЎҢеҸҜиғҪдјҡи®©дҪ йҖүжӢ©дёҖдёӘзј–иҫ‘еҷЁпјҢдёҖиҲ¬йҖү nano е°ұеҫҲеҸӢеҘҪе•ҰпјҒрҹҳҠ

crontab зҡ„ж—¶й—ҙж јејҸжҳҜиҝҷж ·зҡ„пјҡ

жҜҸдёӘеӯ—ж®өзҡ„еҸ–еҖјиҢғеӣҙйғҪиҙҙеҝғең°е‘ҠиҜүдҪ е•Ұпјҡ

  • еҲҶпјҡ 0-59
  • ж—¶пјҡ 0-23
  • ж—Ҙпјҡ 1-31
  • жңҲпјҡ 1-12
  • е‘Ёпјҡ 0-7 (0 е’Ң 7 йғҪиЎЁзӨәжҳҹжңҹеӨ©е“ҰпјҒ)

дёҖдәӣи¶…е®һз”Ёзҡ„е°ҸжҠҖе·§пјҡ

  • *пјҡ иЎЁзӨәиҝҷдёӘеӯ—ж®өзҡ„жҜҸдёҖдёӘеҖјйғҪз¬ҰеҗҲгҖӮжҜ”еҰӮ * * * * * е°ұиЎЁзӨәжҜҸеҲҶй’ҹйғҪжү§иЎҢеҗҺйқўзҡ„е‘Ҫд»ӨпјҒрҹҳұ
  • -пјҡ иЎЁзӨәдёҖдёӘиҝһз»ӯзҡ„иҢғеӣҙгҖӮжҜ”еҰӮ 1-7 е°ұиЎЁзӨә 1 еҲ° 7гҖӮ
  • ,пјҡ иЎЁзӨәеӨҡдёӘдёҚиҝһз»ӯзҡ„еҖјгҖӮжҜ”еҰӮ 1,2,6 е°ұиЎЁзӨә 1гҖҒ2 е’Ң 6гҖӮ
  • /пјҡ иЎЁзӨәжҜҸйҡ”еӨҡд№…жү§иЎҢдёҖж¬ЎгҖӮжҜ”еҰӮ 0/10 еңЁвҖңеҲҶвҖқзҡ„еӯ—ж®өиЎЁзӨәжҜҸйҡ” 10 еҲҶй’ҹжү§иЎҢдёҖж¬ЎпјҲд№ҹе°ұжҳҜ 0, 10, 20, 30, 40, 50 еҲҶпјүгҖӮ

дёҫеҮ дёӘи¶…з®ҖеҚ•зҡ„е®ҡж—¶д»»еҠЎж —еӯҗрҹҢ°пјҡ

  • жҜҸеӨ©еҮҢжҷЁ 3 зӮ№еӨҮд»ҪжҲ‘зҡ„йҮҚиҰҒж–Үд»¶пјҡ 0 3 * * * /bin/bash /path/to/backup_script.sh
  • жҜҸдёӘе·ҘдҪңж—Ҙзҡ„дёӢеҚҲ 5 зӮ№еҸ‘йҖҒдёҖе°ҒйӮ®д»¶жҸҗйҶ’иҮӘе·ұдёӢзҸӯпјҡ 0 17 * * 1-5 /usr/bin/mail -s "дёӢзҸӯе•ҰпјҒ" your_email@example.com < /dev/null
  • жҜҸйҡ” 30 еҲҶй’ҹжЈҖжҹҘдёҖдёӢжңҚеҠЎеҷЁзҠ¶жҖҒпјҡ */30 * * * * /path/to/check_server_status.sh

зј–иҫ‘е®Ң crontab ж–Үд»¶и®°еҫ—дҝқеӯҳе“ҰпјҒз”өи„‘е°ұдјҡд№–д№–ең°жҢүз…§дҪ зҡ„ж—¶й—ҙиЎЁеҺ»е·ҘдҪңе•ҰпјҒжҳҜдёҚжҳҜж„ҹи§үиҮӘе·ұеғҸдёӘе°Ҹе°Ҹзҡ„вҖңж—¶й—ҙйӯ”жі•еёҲвҖқпјҹрҹ§ҷвҖҚвҷҖпёҸвңЁ

еҘҪе•ҰпјҢд»ҠеӨ©е…ідәҺ Linux жңҚеҠЎз®ЎзҗҶе’Ңе®ҡж—¶д»»еҠЎзҡ„е°ҸжҠҖе·§е°ұеҲҶдә«еҲ°иҝҷйҮҢе•ҰпјҒеӯҰдјҡдәҶиҝҷдәӣпјҢдҪ зҡ„ Linux зі»з»ҹе°ұиғҪжӣҙеҠ жҷәиғҪе’ҢиҮӘеҠЁеҢ–е•ҰпјҒдёӢж¬ЎеҶҚе’Ңе…„ејҹд»¬еҲҶдә«жӣҙеӨҡжңүи¶Јзҡ„ Linux зҹҘиҜҶпјҒрҹ’–рҹ’»

06-иҪҜд»¶е®үиЈ…е’Ңзі»з»ҹжңҚеҠЎ

Posted on 2025-04-26

е…„ејҹд»¬пјҢд»ҠеӨ©жқҘиҒҠиҒҠеңЁ Linux зі»з»ҹдёҠжҖҺд№ҲвҖңжҚҜйҘ¬вҖқеҗ„з§ҚиҪҜд»¶е’Ңзі»з»ҹжңҚеҠЎпјҒрҹ’»вңЁ еҲ«зңӢе‘Ҫд»ӨиЎҢй»‘д№Һд№Һзҡ„пјҢзҺ©зҶҹдәҶз®Җзӣҙжү“ејҖж–°дё–з•Ңзҡ„еӨ§й—ЁпјҒрҹҳҺ

иҪҜд»¶еҲҶзұ»пјҡдҪ зҡ„ App вҖңеҮәз”ҹвҖқж–№ејҸеӨ§жҸӯз§ҳпјҒрҹ‘¶вһЎпёҸрҹҡҖ

иҜқиҜҙиҪҜд»¶е°ұеғҸжҲ‘д»¬зҡ„е°ҸеҸҜзҲұ AppпјҢе®ғд»¬зҡ„вҖңеҮәз”ҹвҖқж–№ејҸеҸҜдёҚдёҖж ·е“ҰпјҒдё»иҰҒеҲҶдёӨз§Қпјҡ

1. жәҗз ҒеҢ…пјҡ е°ұеғҸжӢҝеҲ° App зҡ„вҖңжәҗд»Јз ҒвҖқпјҒрҹ“ң дҪ йңҖиҰҒиҮӘе·ұй…ҚзҪ®еҘҪвҖңејҖеҸ‘зҺҜеўғвҖқпјҲеҗ„з§Қдҫқиө–пјүпјҢз„¶еҗҺеғҸдёӘе°Ҹз ҒеҶңдёҖж ·вҖңзј–иҜ‘вҖқе®ғпјҢеҶҚвҖңжү“еҢ…вҖқжҲҗиғҪе®үиЈ…зҡ„вҖңе®үиЈ…еҢ…вҖқпјҢжңҖеҗҺжүҚиғҪе®үиЈ…дҪҝз”ЁгҖӮеҗ¬иө·жқҘжҳҜдёҚжҳҜжңүзӮ№зЎ¬ж ёпјҹрҹ’Ә дҪҶеҘҪеӨ„жҳҜдҪ еҸҜд»Ҙж №жҚ®иҮӘе·ұзҡ„йңҖжұӮе®ҡеҲ¶еҠҹиғҪпјҢе°ұеғҸиҮӘе·ұ DIY дёҖдёӘзӢ¬дёҖж— дәҢзҡ„жүӢжңәеЈіпјҒрҹҺЁ

дёҫдёӘж —еӯҗрҹҢ°пјҡ еҫҲеӨҡи¶… coolгҖҒжңҖж–°зҡ„ејҖжәҗиҪҜд»¶йғҪжҳҜд»Ҙжәҗз ҒеҢ…зҡ„еҪўејҸеҸ‘еёғзҡ„гҖӮжҜ”еҰӮдҪ жғіз”ЁжҹҗдёӘиҝҳжІЎиў«дё»жөҒ Linux еҸ‘иЎҢзүҲж”¶еҪ•зҡ„й»‘з§‘жҠҖе·Ҙе…·пјҢеҫҲеҸҜиғҪе°ұйңҖиҰҒдҪ е…ҲдёӢиҪҪжәҗз ҒпјҢз„¶еҗҺ ./configure -> make -> make install дёүиҝһеҮ»пјҒрҹ’Ҙ

2. дәҢиҝӣеҲ¶еҢ…пјҡ иҝҷе°ұеғҸзӣҙжҺҘжӢҝеҲ°е·Із»Ҹжү“еҢ…еҘҪзҡ„ App е®үиЈ…еҢ…пјҲжҜ”еҰӮе®үеҚ“зҡ„ .apkпјҢiOS зҡ„ .ipaпјүгҖӮдҪ дёҚйңҖиҰҒе…іеҝғе®ғиғҢеҗҺзҡ„вҖңжәҗд»Јз ҒвҖқжҳҜе•ҘпјҢзӣҙжҺҘвҖңдёҖй”®е®үиЈ…вҖқе°ұиғҪз”ЁпјҒвңҢпёҸ иҝҷз§Қж–№ејҸдёәдәҶдҝқжҠӨејҖеҸ‘иҖ…зҡ„зҹҘиҜҶдә§жқғпјҢдјҡиҝӣиЎҢд»Јз Ғж··ж·ҶпјҢи®©еҲ«дәәдёҚе®№жҳ“зӣҙжҺҘзңӢеҲ°жәҗз ҒгҖӮ

дәҢиҝӣеҲ¶еҢ…еҸҲеҲҶзҰ»зәҝе®үиЈ…е’ҢеңЁзәҝе®үиЈ…дёӨз§Қж–№ејҸпјҢеҗ„жңүеҚғз§ӢпјҢйҖӮз”ЁдёҚеҗҢеңәжҷҜе“ҰпјҒ

RPM зҰ»зәҝе®үиЈ…пјҡзҪ‘дёҚеҘҪпјҹжІЎе…ізі»пјҒрҹ“Ұ

RPMпјҲRed Hat Package ManagerпјүжҳҜеҫҲеӨҡ Linux еҸ‘иЎҢзүҲпјҲжҜ”еҰӮ CentOSгҖҒFedoraпјүйғҪеңЁз”Ёзҡ„вҖңеҝ«йҖ’вҖқж јејҸпјҒрҹ“Ұ дҪ еҸҜд»ҘжҸҗеүҚдёӢиҪҪеҘҪ .rpm еҢ…пјҢз„¶еҗҺеңЁжІЎзҪ‘жҲ–иҖ…зҪ‘з»ңдёҚеҘҪзҡ„жғ…еҶөдёӢзӣҙжҺҘе®үиЈ…пјҒ

еёёз”Ёе‘Ҫд»Өе°ҸиҙҙеЈ«пјҡ

  • жҹҘиҜўе·Іе®үиЈ…зҡ„ RPM еҢ…пјҡ rpm -qa пјҲиҝҷдёӘе‘Ҫд»ӨдјҡеҲ—еҮәжүҖжңүе·Іе®үиЈ…зҡ„ RPM еҢ…пјҢдҝЎжҒҜйҮҸжңүзӮ№еӨ§е“ҲпјҒрҹ‘Җпјү
  • зІҫеҮҶжҹҘжүҫеҢ…еҗ«зү№е®ҡе…ій”®иҜҚзҡ„еҢ…пјҡ rpm -qa | grep <е…ій”®иҜҚ> жҜ”еҰӮдҪ жғіжүҫеҢ…еҗ« wget зҡ„еҢ…пјҡrpm -qa | grep wget
  • жҹҘзңӢзү№е®ҡ RPM еҢ…зҡ„иҜҰз»ҶдҝЎжҒҜпјҡ rpm -qi <еҢ…еҗҚ> жҜ”еҰӮпјҡrpm -qi wget-1.21.1-8.el9.x86_64
  • е®үиЈ…жң¬ең° RPM еҢ…пјҡ rpm -ivh <.rpm еҢ…ж–Үд»¶> i жҳҜе®үиЈ…пјҢv жҳҜжҳҫзӨәиҜҰз»ҶдҝЎжҒҜпјҢh жҳҜжҳҫзӨәе®үиЈ…иҝӣеәҰе°Ҹж–№еқ—пјҢж•Іиө·жқҘжҳҜдёҚжҳҜеҫҲжңүд»ӘејҸж„ҹпјҹрҹ§ұ
    • ејәеҲ¶е®үиЈ…пјҲе°ҸеҝғдҪҝз”ЁпјҒпјүпјҡ жңүж—¶еҖҷйҒҮеҲ°дҫқиө–й—®йўҳпјҢеҸҜд»Ҙе°қиҜ•еҠ дёҠ --nodeps --forceпјҢдҪҶеҸҜиғҪдјҡеҜјиҮҙиҪҜд»¶иҝҗиЎҢдёҚзЁіе®ҡе“ҰпјҒвҡ пёҸ
  • еҚҮзә§ RPM еҢ…пјҡ rpm -Uvh <.rpm еҢ…ж–Үд»¶> U жҳҜеҚҮзә§пјҢеҰӮжһңжІЎе®үиЈ…иҝҮе°ұзӣёеҪ“дәҺе®үиЈ…гҖӮ
  • еҚёиҪҪ RPM еҢ…пјҡ rpm -e <еҢ…еҗҚ> жіЁж„ҸиҝҷйҮҢжҳҜеҢ…еҗҚпјҢдёҚжҳҜ .rpm ж–Үд»¶еҗҚпјҒжҜ”еҰӮеҚёиҪҪ wgetпјҡrpm -e wget
    • ејәеҲ¶еҚёиҪҪпјҲеҗҢж ·е°ҸеҝғпјҒпјүпјҡ еҗҢж ·еҸҜд»ҘеҠ дёҠ --nodeps --forceпјҢдҪҶеҗҺжһңиҮӘиҙҹе“ҰпјҒрҹ’Ј

DNF еңЁзәҝе®үиЈ…пјҡеҶІжөӘиҫҫдәәзҡ„еҝ…еӨҮжҠҖиғҪпјҒрҹҸ„вҖҚвҷҖпёҸ

DNFпјҲDandified YUMпјүжҳҜж–°дёҖд»Јзҡ„еҢ…з®ЎзҗҶеҷЁпјҢеңЁ Fedora е’ҢдёҖдәӣж–°зҡ„ CentOS зүҲжң¬дёҠеҸ–д»ЈдәҶ YUMгҖӮе®ғе°ұеғҸдёҖдёӘжҷәиғҪзҡ„вҖңApp StoreвҖқпјҢиҝһжҺҘзқҖиҪҜд»¶д»“еә“пјҢеҸҜд»ҘиҮӘеҠЁеё®дҪ дёӢиҪҪгҖҒе®үиЈ…гҖҒеҚҮзә§гҖҒеҚёиҪҪиҪҜд»¶пјҢиҝҳиғҪиҮӘеҠЁеӨ„зҗҶдҫқиө–е…ізі»пјҢз®ҖзӣҙжҳҜжҮ’дәәеҝ…еӨҮпјҒрҹҳҺ

еёёз”Ёе‘Ҫд»ӨйҖҹи®°пјҡ

  • еҲ—еҮәжүҖжңүеҸҜз”Ёзҡ„иҪҜд»¶еҢ…пјҡ dnf list пјҲдҝЎжҒҜйҮҸе·ЁеӨ§пјҒрҹҗіпјү
  • жҗңзҙўиҪҜд»¶еҢ…пјҡ dnf search <е…ій”®иҜҚ> жҜ”еҰӮдҪ жғіжүҫе…ідәҺ Python зҡ„еҢ…пјҡdnf search python
  • жҹҘзңӢиҪҜд»¶еҢ…дҝЎжҒҜпјҡ dnf info <еҢ…еҗҚ> жҜ”еҰӮпјҡdnf info python3
  • е®үиЈ…иҪҜд»¶еҢ…пјҡ dnf install <еҢ…еҗҚ> жҜ”еҰӮе®үиЈ… Pythonпјҡdnf install python3 е®ғдјҡиҮӘеҠЁеё®дҪ дёӢиҪҪе№¶е®үиЈ…жүҖжңүзӣёе…ізҡ„дҫқиө–еҢ…пјҢи¶…зңҒеҝғпјҒрҹ’–
  • еҚҮзә§иҪҜд»¶еҢ…пјҡ dnf upgrade <еҢ…еҗҚ> еҚҮзә§жҢҮе®ҡзҡ„еҢ…пјҢжҜ”еҰӮеҚҮзә§ Pythonпјҡdnf upgrade python3
  • еҚҮзә§жүҖжңүиҪҜд»¶еҢ…пјҡ dnf update жҲ–иҖ… dnf upgrade пјҲжҺЁиҚҗе®ҡжңҹжү§иЎҢпјҢдҝқжҢҒзі»з»ҹиҪҜд»¶жңҖж–°пјҒвңЁпјү
  • з§»йҷӨиҪҜд»¶еҢ…пјҡ dnf remove <еҢ…еҗҚ> жҜ”еҰӮз§»йҷӨ Pythonпјҡdnf remove python3
    • з§»йҷӨдёҚеҶҚйңҖиҰҒзҡ„дҫқиө–еҢ…пјҡ dnf autoremove еҚёиҪҪжҺүйӮЈдәӣеӣ дёәд№ӢеүҚе®үиЈ…жҹҗдёӘеҢ…иҖҢиҮӘеҠЁе®үиЈ…пјҢдҪҶзҺ°еңЁйӮЈдёӘеҢ…е·Із»Ҹиў«еҚёиҪҪдәҶзҡ„вҖңеӯӨе„ҝвҖқдҫқиө–еҢ…пјҢи®©дҪ зҡ„зі»з»ҹжӣҙе№ІеҮҖпјҒрҹ§№
  • еҲ—еҮәе·Іе®үиЈ…зҡ„иҪҜд»¶еҢ…пјҡ dnf list --installed жғізңӢзңӢиҮӘе·ұйғҪиЈ…дәҶе•Ҙпјҹз”ЁиҝҷдёӘпјҒрҹ§җ
  • жҹҘжүҫе·Іе®үиЈ…зҡ„еҢ…еҗ«зү№е®ҡе…ій”®иҜҚзҡ„еҢ…пјҡ dnf list --installed | grep <е…ій”®иҜҚ> жҜ”еҰӮжҹҘжүҫе·Іе®үиЈ…зҡ„ Python еҢ…пјҡdnf list --installed | grep python
  • жё…зҗҶзј“еӯҳпјҡ dnf clean all жңүж—¶еҖҷйҒҮеҲ°й—®йўҳеҸҜд»Ҙе°қиҜ•жё…зҗҶдёҖдёӢзј“еӯҳгҖӮ

дёҫеҮ дёӘ DNF зҡ„е®һз”ЁдҫӢеӯҗрҹҢ°пјҡ

  • жғіе®үиЈ…дёҖдёӘи¶…еҘҪз”Ёзҡ„ж–Үжң¬зј–иҫ‘еҷЁ nanoпјҡ sudo dnf install nano
  • еҸ‘зҺ° vim еӨӘйҡҫеӯҰпјҢжғіжҚўеӣһ nanoпјҹ sudo dnf remove vim
  • еҗ¬иҜҙ neofetch еҸҜд»Ҙй…·зӮ«ең°еұ•зӨәзі»з»ҹдҝЎжҒҜпјҢжғіиҜ•иҜ•пјҹ sudo dnf install neofetch з„¶еҗҺиҝҗиЎҢ neofetch зңӢзңӢж•ҲжһңпјҒрҹӨ©
  • зі»з»ҹжҸҗзӨәжңүжӣҙж–°пјҢжғідёҖж¬ЎжҖ§еҚҮзә§жүҖжңүиҪҜд»¶пјҹ sudo dnf update -y пјҲеҠ дёҠ -y еҸҜд»ҘиҮӘеҠЁзЎ®и®ӨпјҢдёҚз”ЁжҜҸж¬ЎйғҪжҢү y е•ҰпјҒрҹҳүпјү

еҘҪе•ҰпјҢе…ідәҺ Linux дёӢзҡ„иҪҜд»¶е®үиЈ…е°ұе…ҲиҒҠеҲ°иҝҷйҮҢпјҒжҳҜдёҚжҳҜж„ҹи§үд№ҹжІЎйӮЈд№Ҳйҡҫеҳӣпјҹрҹҳү жҺҢжҸЎдәҶиҝҷдәӣе°ҸжҠҖе·§пјҢдҪ д№ҹиғҪиҪ»жқҫз®ЎзҗҶдҪ зҡ„ Linux зі»з»ҹе•ҰпјҒдёӢж¬ЎеҶҚе’Ңе…„ејҹд»¬еҲҶдә«жӣҙеӨҡеҘҪзҺ©зҡ„жҠҖжңҜе№Іиҙ§пјҒрҹ’–вңЁ

05-жқғйҷҗз®ЎзҗҶ

Posted on 2025-04-26

рҹҢҹ Linux жқғйҷҗз®ЎзҗҶж–°жүӢе°ҸзҷҪд№ҹиғҪзңӢжҮӮпјҒе‘ҠеҲ«ж–Үд»¶вҖңй”Ғрҹ”’вҖқзғҰжҒјпјҒ

е…„ејҹд»¬пјҢд»ҠеӨ©з»ҷеӨ§е®¶еҲҶдә«дёҖдёӢеңЁ Linux зі»з»ҹйҮҢи¶…зә§йҮҚиҰҒзҡ„дёҖдёӘзҹҘиҜҶзӮ№вҖ”вҖ”жқғйҷҗз®ЎзҗҶпјҒрҹ’»рҹ’ҫ еҲ«зңӢе®ғеҘҪеғҸеҫҲй«ҳж·ұзҡ„ж ·еӯҗпјҢе…¶е®һжҺҢжҸЎдәҶе®ғпјҢе°ұиғҪжӣҙеҘҪең°дҝқжҠӨжҲ‘д»¬зҡ„ж–Үд»¶е’Ңж•°жҚ®е“ҰпјҒрҹ’Ә

рҹӨ” жқғйҷҗжҳҜдёӘе•Ҙпјҹ

еңЁ Linux иҝҷдёӘзҘһеҘҮзҡ„дё–з•ҢйҮҢпјҢжҜҸдёӘж–Үд»¶е’Ңж–Үд»¶еӨ№йғҪжңүиҮӘе·ұзҡ„вҖңдё»дәәвҖқе’ҢвҖңз®Ўе®¶вҖқпјҢиҝҳжңүдёҖдәӣвҖңи®ҝе®ўвҖқгҖӮжқғйҷҗе°ұеғҸжҳҜз»ҷд»–д»¬и®ҫзҪ®зҡ„вҖңйҖҡиЎҢиҜҒвҖқпјҢи§„е®ҡдәҶи°ҒиғҪзңӢрҹ‘ҖгҖҒи°ҒиғҪеҠЁвңҚпёҸгҖҒи°ҒиғҪз”Ёв–¶пёҸпјҒ

з®ҖеҚ•жқҘиҜҙпјҢжқғйҷҗе°ұдёүз§Қпјҡ

  • r (Read - иҜ»)пјҡрҹ‘Җ еҸҜд»ҘзңӢйҮҢйқўзҡ„еҶ…е®№пјҒ
    • ж–Үд»¶пјҡе°ұиғҪжү“ејҖзңӢеҲ°ж–Үд»¶еҶҷдәҶе•ҘгҖӮ
    • зӣ®еҪ•пјҡе°ұиғҪзңӢеҲ°иҝҷдёӘж–Үд»¶еӨ№йҮҢйғҪжңүе“Әдәӣж–Үд»¶е’Ңж–Үд»¶еӨ№зҡ„еҗҚеӯ—гҖӮ
  • w (Write - еҶҷ)пјҡвңҚпёҸ еҸҜд»Ҙдҝ®ж”№йҮҢйқўзҡ„еҶ…е®№пјҒ
    • ж–Үд»¶пјҡе°ұиғҪзј–иҫ‘гҖҒдҝқеӯҳж–Үд»¶йҮҢзҡ„дёңиҘҝгҖӮ
    • зӣ®еҪ•пјҡе°ұиғҪеңЁиҝҷдёӘж–Үд»¶еӨ№йҮҢж–°е»әгҖҒеҲ йҷӨгҖҒз§»еҠЁж–Үд»¶е’Ңж–Үд»¶еӨ№пјҒвҡ пёҸ е°Ҹеҝғж“ҚдҪңе“ҰпјҒ
  • x (Execute - жү§иЎҢ)пјҡв–¶пёҸ еҸҜд»ҘиҝҗиЎҢе®ғпјҒ
    • ж–Үд»¶пјҡеҜ№дәҺзЁӢеәҸжҲ–иҖ…и„ҡжң¬ж–Үд»¶пјҢжңүдәҶиҝҷдёӘжқғйҷҗжүҚиғҪи·‘иө·жқҘпјҒ
    • зӣ®еҪ•пјҡжңүдәҶиҝҷдёӘжқғйҷҗжүҚиғҪз”Ё cd е‘Ҫд»Өиҝӣе…ҘиҝҷдёӘж–Үд»¶еӨ№пјҒрҹҡӘ

рҹ”Қ жқғйҷҗдҝЎжҒҜеңЁе“ӘйҮҢзңӢпјҹ

еңЁз»Ҳз«ҜйҮҢиҫ“е…Ҙ ls -l е‘Ҫд»ӨпјҢе°ұиғҪзңӢеҲ°ж–Үд»¶е’Ңж–Үд»¶еӨ№зҡ„иҜҰз»ҶдҝЎжҒҜе•ҰпјҒе°ұеғҸиҝҷж ·пјҡ

жңҖеүҚйқўйӮЈдёҖдёІеғҸ drwxr-xr-x е’Ң -rw-r--r-- зҡ„е°ұжҳҜжқғйҷҗдҝЎжҒҜе•ҰпјҒрҹӨ©

  • з¬¬дёҖдҪҚпјҡжҳҜж–Үд»¶зұ»еһӢе“ҰпјҒ- д»ЈиЎЁжҷ®йҖҡж–Үд»¶пјҢd д»ЈиЎЁж–Үд»¶еӨ№пјҢl д»ЈиЎЁеҝ«жҚ·ж–№ејҸгҖӮ

  • еҗҺйқўд№қдҪҚпјҡеҲҶжҲҗдёүз»„пјҢжҜҸз»„дёүдёӘеӯ—жҜҚпјҢеҲҶеҲ«д»ЈиЎЁ ж–Үд»¶дё»дәә (User)гҖҒж–Үд»¶жүҖеұһз»„ (Group)гҖҒе…¶д»–дәә (Others) зҡ„жқғйҷҗпјҒ

вңҚпёҸ жҖҺд№Ҳдҝ®ж”№жқғйҷҗпјҹдёӨз§Қж–№жі•е®үжҺ’дёҠпјҒ

рҹ…°пёҸ еӯ—жҜҚеӨ§жі•пјҡз®ҖеҚ•жҳ“жҮӮпјҒ

з”Ё chmod е‘Ҫд»Өе°ұеҸҜд»Ҙдҝ®ж”№жқғйҷҗе•ҰпјҒ

  • uпјҡдё»дәә
  • gпјҡжүҖеұһз»„
  • oпјҡе…¶д»–дәә
  • +пјҡж·»еҠ жқғйҷҗ
  • -пјҡз§»йҷӨжқғйҷҗ
  • =пјҡи®ҫзҪ®жҲҗжҹҗдёӘжқғйҷҗ

жҜ”еҰӮпјҢжҲ‘жғіз»ҷжҲ‘еҶҷзҡ„ linux.txt ж–Үд»¶еҠ дёҠжү§иЎҢжқғйҷҗпјҢи®©е®ғи·‘иө·жқҘпјҒрҹҸғвҖҚвҷҖпёҸ

еҰӮжһңжғіз»ҷдёҖдёӘж–Үд»¶еӨ№е’Ңе®ғйҮҢйқўзҡ„жүҖжңүдёңиҘҝйғҪеҠ дёҠжҹҗдёӘжқғйҷҗпјҢи®°еҫ—еҠ дёҠ -R еҸӮж•°е“ҰпјҒрҹ”„

рҹ”ў ж•°еӯ—еҜҶз Ғпјҡжӣҙеҝ«жӣҙзӣҙжҺҘпјҒ

иҝҳжңүдёҖз§Қжӣҙй…·зҡ„ж–№жі•пјҢз”Ёж•°еӯ—жқҘд»ЈиЎЁжқғйҷҗпјҒрҹҳҺ

  • r = 4
  • w = 2
  • x = 1

жҠҠжғіиҰҒзҡ„жқғйҷҗж•°еӯ—еҠ иө·жқҘе°ұеҘҪе•ҰпјҒ

  • 7 (4+2+1) = rwx (иҜ»еҶҷжү§иЎҢе…ЁйғҪжңүпјҒ)
  • 6 (4+2) = rw- (еҸҜд»ҘиҜ»еҶҷпјҢдёҚиғҪжү§иЎҢ)
  • 5 (4+1) = r-x (еҸҜд»ҘиҜ»е’Ңжү§иЎҢпјҢдёҚиғҪеҶҷ)
  • 4 = r-- (еҸӘиғҪиҜ»)

жҜ”еҰӮпјҢжҲ‘жғіз»ҷ shop иҝҷдёӘж–Үд»¶еӨ№жүҖжңүжқғйҷҗпјҒе®үжҺ’пјҒрҹ‘Ү

вҡ пёҸ 777 жқғйҷҗиҰҒи°Ёж…ҺдҪҝз”Ёе“ҰпјҢеӣ дёәе®ғз»ҷжүҖжңүдәәејҖдәҶеӨ§й—ЁпјҒрҹҡӘ

рҹӨ« й»ҳи®Өжқғйҷҗзҡ„е°Ҹз§ҳеҜҶ (umask)

жҜҸж¬ЎжҲ‘д»¬ж–°е»әж–Үд»¶жҲ–иҖ…ж–Үд»¶еӨ№зҡ„ж—¶еҖҷпјҢзі»з»ҹйғҪдјҡз»ҷе®ғд»¬дёҖдёӘй»ҳи®Өзҡ„жқғйҷҗгҖӮиҝҷдёӘй»ҳи®ӨжқғйҷҗжҳҜз”ұ umask иҝҷдёӘе°Ҹе®¶дјҷеҶіе®ҡзҡ„пјҒрҹӨ« иҷҪз„¶з¬”и®°йҮҢжІЎз»ҶиҜҙпјҢдҪҶи®°дҪҸжңүиҝҷд№ҲдёӘдёңиҘҝе°ұеҘҪе•ҰпјҒ

вңЁ зү№ж®ҠжқғйҷҗдәҶи§ЈдёҖдёӢпјҒ

Linux иҝҳжңүдёҖдәӣзү№ж®Ҡзҡ„вҖңе°ҸеҪ©иӣӢвҖқжқғйҷҗпјҢиҷҪз„¶дёҚеёёз”ЁпјҢдҪҶжҳҜдәҶи§ЈдёҖдёӢжҖ»жІЎй”ҷпјҒ

  • еҶ’йҷ©дҪҚ (SUID & SGID)пјҡрҹҡҖ и®©жҷ®йҖҡз”ЁжҲ·д№ҹиғҪдёҙж—¶жӢҘжңүж–Үд»¶дё»дәәжҲ–иҖ…ж–Үд»¶жүҖеұһз»„зҡ„вҖңи¶…иғҪеҠӣвҖқпјҒ
    • SпјҡиЎЁзӨәи®ҫзҪ®дәҶеҶ’йҷ©дҪҚпјҢдҪҶдё»дәә/з»„жІЎжңүжү§иЎҢжқғйҷҗпјҲдёҚеёёи§ҒпјүгҖӮ
    • sпјҡиЎЁзӨәи®ҫзҪ®дәҶеҶ’йҷ©дҪҚпјҢе№¶дё”дё»дәә/з»„жңүжү§иЎҢжқғйҷҗпјҲжӣҙеёёи§ҒпјүгҖӮ
  • зІҳж»һдҪҚ (Sticky Bit)пјҡрҹ“Қ з»ҷе…¬е…ұж–Үд»¶еӨ№еҠ дёӘвҖңе°Ҹй—ЁеҚ«вҖқпјҢеҸӘжңүж–Үд»¶зҡ„дё»дәәгҖҒж–Үд»¶еӨ№зҡ„дё»дәәжҲ–иҖ… root еӨ§дҪ¬жүҚиғҪеҲ йҷӨйҮҢйқўзҡ„ж–Үд»¶пјҢеҲ«дәәжғіеҠЁпјҹжІЎй—ЁпјҒрҹҷ…вҖҚвҷҖпёҸ з”Ё T иЎЁзӨәе“ҰпјҒ

рҹӨқ ж–Үд»¶е’Ңж–Үд»¶еӨ№зҡ„вҖңжҲ·еҸЈжң¬вҖқ

  • жүҖеұһз”ЁжҲ· (chown)пјҡеҸҜд»Ҙдҝ®ж”№ж–Үд»¶е’Ңж–Үд»¶еӨ№зҡ„вҖңдё»дәәвҖқпјҒрҹҸ 
  • жүҖеұһз»„ (chgrp)пјҡеҸҜд»Ҙдҝ®ж”№ж–Үд»¶е’Ңж–Үд»¶еӨ№зҡ„вҖңе®¶еәӯвҖқпјҒрҹҸҳпёҸ

рҹӣЎпёҸ жӣҙй«ҳзә§зҡ„жқғйҷҗз®ЎзҗҶ (ACL)

еҰӮжһңи§үеҫ—еҹәжң¬жқғйҷҗдёҚеӨҹз”ЁпјҢиҝҳжңүжӣҙеҺүе®ізҡ„ ACL (Access Control List)пјҒе®ғеҸҜд»Ҙй’ҲеҜ№** РәРҫРҪРәСҖРөСӮРҪСӢР№ з”ЁжҲ·** и®ҫзҪ®жӣҙз»ҶиҮҙзҡ„жқғйҷҗпјҒрҹ“қ е°ұеғҸз»ҷ VIP з”ЁжҲ·ејҖйҖҡдё“еұһйҖҡйҒ“пјҒдёӢж¬ЎжңүжңәдјҡеҶҚз»ҷеӨ§е®¶з»Ҷи®ІпјҒ

рҹ§‘вҖҚрҹӨқвҖҚрҹ§‘ з”ЁжҲ·е’Ңз”ЁжҲ·з»„е°ҸиҜҫе Ӯ

  • groupadd з”ЁжҲ·з»„еҗҚз§°пјҡж–°е»әдёҖдёӘвҖңе®¶еәӯвҖқпјҒ
  • groupmod з”ЁжҲ·з»„еҗҚз§°пјҡдҝ®ж”№вҖңе®¶еәӯвҖқдҝЎжҒҜгҖӮ
  • groupdel з”ЁжҲ·з»„еҗҚз§°пјҡи§Јж•ЈвҖңе®¶еәӯвҖқгҖӮ
  • /etc/groupпјҡиҝҷйҮҢи®°еҪ•дәҶжүҖжңүвҖңе®¶еәӯвҖқзҡ„дҝЎжҒҜе“ҰпјҒ
  • useradd з”ЁжҲ·еҗҚз§°пјҡж–°ж·»дёҖдёӘвҖңе®¶еәӯжҲҗе‘ҳвҖқпјҒ
  • usermod з”ЁжҲ·еҗҚз§°пјҡдҝ®ж”№вҖңе®¶еәӯжҲҗе‘ҳвҖқдҝЎжҒҜгҖӮ
