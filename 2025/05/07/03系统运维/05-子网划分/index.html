05-子网划分

Posted on 2025-05-07 Edited on 2025-05-09

子网掩码 —-根据ip地址和子网掩码 可以划分子网及子网范围

1 由连续1+连续的0 组成的
2 网络地址的所有位置都为1，对应于主机地址的所有位置都为0。
3 划分成网络地址和主机地址

网络位对应的二进制即为: 上下相与的结果就是网络位.

如何划分A,B,C类网络 及公网和私网的区别

A类IP的网络位 必须是 0          0

 B类IP的网络位  必须是 10       128

 C类IP的网络位  必须是  110     192
范围 (公网地址) 分类 私有 IP 地址范围 网络类型
0.0.0.0 ~ 127.255.255.255 A 类 10.0.0.0 ~ 10.255.255.255 大规模
128.0.0.0 ~ 191.255.255.255 B 类 172.16.0.0 ~ 172.31.255.255 中等规模
192.0.0.0 ~ 223.255.255.255 C 类 192.168.0.0 ~ 192.168.255.255 小规模

技能: 判断链两个ip地址是否处于同一个网段

子网划分

IP地址=网络位+子网位+主机位
n: 需要借用多少个主机位作为子网位，计算公式：2^n >= 需要子网数
m: 子网需要的主机个数， 计算公式： 2^m-2 >= 需要主机个数
