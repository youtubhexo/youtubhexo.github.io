💻 效率翻倍！SSH 服务那些你不知道的小技巧！

Posted on 2025-04-29

兄弟们，今天给大家分享一些关于 SSH 服务的实用小技巧，让你的开发效率直线 Up！🚀

虚拟机多开大法！💻💻💻

还在一个一个打开虚拟机？No No No！试试扫描你的虚拟机管理工具，很多都支持一次性启动多个虚拟机哦！解放双手，节省时间，简直不要太爽！

ISO 文件用完就扔？👋

系统安装好之后，那个 .iso 镜像文件就可以功成身退啦！删掉它，释放你的磁盘空间！

SSH 认证小秘密🔑

你知道 SSH 登录的时候，会先检查服务器的公钥文件里有没有你的记录吗？就像门卫大爷认门一样，认证通过才能顺利进入！

服务状态随时掌握！👀

想知道 SSH 服务有没有在好好工作？用 service ssh status 或者更简洁的 ps aux | grep sshd 都可以查看服务状态哦！

配置文件别乱动！🙅‍♀️

SSH 的配置文件一般在 /etc/ssh/ 目录下，姐妹们千万不要直接修改原始文件！想要自定义配置，可以新建一个配置文件，这样更安全也更方便管理。

远程操作小贴士！💡

用 SSH 远程登录到另一台机器后，想退出来？直接输入 exit 就行啦！跟 Ctrl + C 的效果差不多哦！

文件传输 So Easy！💨

  • scp大法好！ 本地文件想传到远程服务器？一条命令搞定！远程的文件想下载到本地？也是一样简单！

    • 本地复制到远程：scp your_local_file user@remote_ip:/path/to/remote_directory
    • 远程复制到本地：scp user@remote_ip:/path/to/remote_file /path/to/local_directory

  • 告别垃圾同步！ 某度网盘的同步简直让人抓狂？试试 rsync 命令吧！它可以实现更高效、更可靠的文件同步！

自动化运维小助手！🤖

基于 SSH，我们可以实现很多自动化操作，比如自动发送命令到服务器。不过要注意，有些涉及到进程的命令（比如 source）可能会执行不成功哦！

-t 参数了解一下！😉

在使用 ssh root@192.168.88.102 连接服务器的时候，强烈推荐加上 -t 参数！你会看到一个友好的提示信息：Connection to 192.168.88.101，让你更清楚连接状态！

SSH 的无限可能！✨

基于 SSH，我们可以做的事情真的太多啦！比如用 scp 把文件轻松复制到不同的服务器，简直是运维小姐姐的必备技能！

Vim 编辑小技巧！✍️

  • 在 Vim 编辑器里，想要一次性删除多行？试试 5dd 吧！数字 5 可以替换成你想要删除的行数哦！
  • 不小心操作失误？别慌！输入 u 就可以撤销上一步操作啦！

公钥文件小知识！🔑

记住哦，每一个 SSH 连接都会在你的 ~/.ssh/authorized_keys 公钥文件里占用一行记录！

希望这些小技巧能帮助到大家！学会这些，你也能成为高效运维小能手！💪 还有什么关于 SSH 的问题，欢迎在评论区留言交流哦！💖
