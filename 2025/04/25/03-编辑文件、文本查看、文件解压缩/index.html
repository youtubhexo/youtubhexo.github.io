03-文件编辑、文本查看与解压缩：我的 Linux 工具箱 🧰

Posted on 2025-04-25

文件编辑：Vim 初体验 📝

Vim 是 Linux 下强大的文本编辑器，虽然上手可能需要一些时间，但熟练后效率很高。

  • 打开文件：

    • vim <文件名>：打开指定文件。如果文件不存在，则会创建一个新文件。
    • vim +<行号> <文件名>：打开文件并将光标定位到指定行号。（例如 vim +10 hello.txt 会打开 hello.txt 并跳转到第 10 行。）

  • 三种模式：Vim 的灵魂 🎭

    • 命令模式 (Normal mode)：Vim 启动后的默认模式，用于执行各种编辑命令（移动光标、复制粘贴、删除等）。

      • yy：复制当前行（yank）。
      • G：跳转到文件末尾。
      • o：在当前行下方插入新行并切换到插入模式。
      • 其他常用命令：（思考：还有哪些常用的命令模式操作，例如 dd (删除行), p (粘贴), hjkl (光标移动) 等？）

    • 插入模式 (Insert mode)：用于输入文本。

      • 从命令模式切换到插入模式的常用按键：i (在光标前插入), a (在光标后插入), o (在当前行下方插入新行), I (在行首插入), A (在行尾插入), O (在当前行上方插入新行)。
      • 按 Esc 键返回命令模式。

    • 末行模式 (Command-line mode)：在命令模式下输入 : 进入，用于执行保存、退出、查找、设置等操作。

      • :q!：强制退出，不保存任何修改。（小心使用！）
      • :x：保存并退出（只有在文件内容被修改时才保存，运维常用！）。
      • :wq：保存并退出（强制保存并退出）。
      • :set nu：设置显示行号。
      • :set nonu：取消显示行号。
      • /<关键词>：在文件中查找关键词，按 n 查找下一个匹配结果，按 N 查找上一个匹配结果。

文本查看命令：洞察文件内容 👀

这些命令用于查看文本文件的内容，各有特点：

  • cat <文件>：一次性显示文件所有内容。（适用于小文件。）
  • more <文件>：分页显示文件内容。
    • 空格：向下翻一页。
    • b：向上翻一页。
    • /关键词：查找关键词。
    • q：退出。
    • 显示当前进度。
  • less <文件>：更强大的分页查看器，功能比 more 更丰富。
    • 空格：向下翻一页。
    • b：向上翻一页。
    • /关键词：查找关键词，按 n 查看下一个匹配，N 查看上一个匹配。
    • q：退出。
    • 支持上下方向键逐行滚动。
  • head -n <文件>：显示文件的前 n 行（默认显示前 10 行）。（例如 head -20 hello.txt 显示前 20 行。）
  • tail -n <文件>：显示文件的后 n 行（默认显示后 10 行）。
  • tail -f <文件>：实时监控文件尾部的更新，常用于查看日志文件。（当你需要实时查看日志变化时非常有用。）
  • wc <文件>：统计文件的行数、单词数和字节数（Word Count）。
    • -l：统计行数（lines）。
    • -w：统计单词数（words）。
    • -c：统计字节数（bytes）。
    • -m：统计字符数（characters）。
  • grep <要查找的内容> <文件名>：强大的文本搜索工具，用于在文件中查找包含指定模式的行。（思考：grep 还有哪些常用的选项，例如 -i (忽略大小写), -r (递归搜索), -v (反向匹配) 等？）

文件解压缩：整理你的文件 📦

在 Linux 上处理压缩文件是很常见的操作。

  • tar 命令 (归档和压缩/解压缩)：常用于打包文件，并结合 gzip 或 bzip2 进行压缩。

    • 打包并使用 gzip 压缩 (.tar.gz 或 .tgz)：
      tar -zcvf <归档文件名>.tar.gz <需要归档的目录/文件>

      • -z：使用 gzip 进行压缩。
      • -c：创建归档文件。
      • -v：显示详细的归档过程（verbose）。
      • -f <归档文件名>：指定归档文件名。
      • <需要归档的目录/文件>：要打包的文件或目录列表。
      • -C <目标目录>：改变到指定的目录 <目标目录> 再执行压缩。（注意：这个 -C 选项的位置可能会影响理解，通常在解压时使用较多。）

    • 解压 .tar.gz 文件：
      tar -zxvf <归档文件名>.tar.gz -C <解压到的目录>

      • -z：使用 gzip 进行解压缩。
      • -x：解压归档文件。
      • -v：显示详细的解压过程。
      • -f <归档文件名>：指定要解压的归档文件名。
      • -C <解压到的目录>：指定解压的目标目录。（例如 -C /root 表示解压到 /root 目录下。）

  • zip 命令 (压缩和解压缩)：常见的跨平台压缩格式。

    • 压缩文件或目录：
      zip -r <归档文件名>.zip <需要归档的目录/文件列表>

      • -r：递归压缩目录。
      • <归档文件名>.zip：指定的压缩文件名。
      • <需要归档的目录/文件列表>：要压缩的文件或目录。

    • 解压 .zip 文件：
      unzip <归档文件名>.zip -d <解压到的目录>

      • <归档文件名>.zip：要解压的 zip 文件。
      • -d <解压到的目录>：指定解压的目标目录。
