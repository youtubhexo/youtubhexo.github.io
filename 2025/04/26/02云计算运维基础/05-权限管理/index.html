05-权限管理

Posted on 2025-04-26

🌟 Linux 权限管理新手小白也能看懂！告别文件“锁🔒”烦恼！

兄弟们，今天给大家分享一下在 Linux 系统里超级重要的一个知识点——权限管理！💻💾 别看它好像很高深的样子，其实掌握了它，就能更好地保护我们的文件和数据哦！💪

🤔 权限是个啥？

在 Linux 这个神奇的世界里，每个文件和文件夹都有自己的“主人”和“管家”，还有一些“访客”。权限就像是给他们设置的“通行证”，规定了谁能看👀、谁能动✍️、谁能用▶️！

简单来说，权限就三种：

  • r (Read - 读)：👀 可以看里面的内容！
    • 文件：就能打开看到文件写了啥。
    • 目录：就能看到这个文件夹里都有哪些文件和文件夹的名字。
  • w (Write - 写)：✍️ 可以修改里面的内容！
    • 文件：就能编辑、保存文件里的东西。
    • 目录：就能在这个文件夹里新建、删除、移动文件和文件夹！⚠️ 小心操作哦！
  • x (Execute - 执行)：▶️ 可以运行它！
    • 文件：对于程序或者脚本文件，有了这个权限才能跑起来！
    • 目录：有了这个权限才能用 cd 命令进入这个文件夹！🚪

🔍 权限信息在哪里看？

在终端里输入 ls -l 命令，就能看到文件和文件夹的详细信息啦！就像这样：

最前面那一串像 drwxr-xr-x 和 -rw-r--r-- 的就是权限信息啦！🤩

  • 第一位：是文件类型哦！- 代表普通文件，d 代表文件夹，l 代表快捷方式。

  • 后面九位：分成三组，每组三个字母，分别代表 文件主人 (User)、文件所属组 (Group)、其他人 (Others) 的权限！

✍️ 怎么修改权限？两种方法安排上！

🅰️ 字母大法：简单易懂！

用 chmod 命令就可以修改权限啦！

  • u：主人
  • g：所属组
  • o：其他人
  • +：添加权限
  • -：移除权限
  • =：设置成某个权限

比如，我想给我写的 linux.txt 文件加上执行权限，让它跑起来！🏃‍♀️

如果想给一个文件夹和它里面的所有东西都加上某个权限，记得加上 -R 参数哦！🔄

🔢 数字密码：更快更直接！

还有一种更酷的方法，用数字来代表权限！😎

  • r = 4
  • w = 2
  • x = 1

把想要的权限数字加起来就好啦！

  • 7 (4+2+1) = rwx (读写执行全都有！)
  • 6 (4+2) = rw- (可以读写，不能执行)
  • 5 (4+1) = r-x (可以读和执行，不能写)
  • 4 = r-- (只能读)

比如，我想给 shop 这个文件夹所有权限！安排！👇

⚠️ 777 权限要谨慎使用哦，因为它给所有人开了大门！🚪

🤫 默认权限的小秘密 (umask)

每次我们新建文件或者文件夹的时候，系统都会给它们一个默认的权限。这个默认权限是由 umask 这个小家伙决定的！🤫 虽然笔记里没细说，但记住有这么个东西就好啦！

✨ 特殊权限了解一下！

Linux 还有一些特殊的“小彩蛋”权限，虽然不常用，但是了解一下总没错！

  • 冒险位 (SUID & SGID)：🚀 让普通用户也能临时拥有文件主人或者文件所属组的“超能力”！
    • S：表示设置了冒险位，但主人/组没有执行权限（不常见）。
    • s：表示设置了冒险位，并且主人/组有执行权限（更常见）。
  • 粘滞位 (Sticky Bit)：📍 给公共文件夹加个“小门卫”，只有文件的主人、文件夹的主人或者 root 大佬才能删除里面的文件，别人想动？没门！🙅‍♀️ 用 T 表示哦！

🤝 文件和文件夹的“户口本”

  • 所属用户 (chown)：可以修改文件和文件夹的“主人”！🏠
  • 所属组 (chgrp)：可以修改文件和文件夹的“家庭”！🏘️

🛡️ 更高级的权限管理 (ACL)

如果觉得基本权限不够用，还有更厉害的 ACL (Access Control List)！它可以针对** конкретный 用户** 设置更细致的权限！📝 就像给 VIP 用户开通专属通道！下次有机会再给大家细讲！

🧑‍🤝‍🧑 用户和用户组小课堂

  • groupadd 用户组名称：新建一个“家庭”！
  • groupmod 用户组名称：修改“家庭”信息。
  • groupdel 用户组名称：解散“家庭”。
  • /etc/group：这里记录了所有“家庭”的信息哦！
  • useradd 用户名称：新添一个“家庭成员”！
  • usermod 用户名称：修改“家庭成员”信息。
  • userdel 用户名称：移除“家庭成员”。

好啦！今天的 Linux 权限管理小课堂就到这里啦！是不是感觉没那么难了呢？😜 掌握了这些，就能更好地管理你的 Linux 系统，保护好你的数据哦！快去试试看吧！👋
