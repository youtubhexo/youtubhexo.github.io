01-python基本语法

Posted on 2025-04-26

姐妹们，今天来聊聊让效率飞升的秘密武器！🚀✨ 没错，就是我们程序员界人见人爱的 Python！🐍💖 听说学会它，能让你早点下班，多追几集剧哦！😉

磨刀不误砍柴工！先搞定基础装备！🛠️

在开始我们酷炫的 Python 之旅前，先要准备好我们的“装备”！就像化妆前要先护肤一样重要哦！😉

  • 基础办公软件安装： 这个嘛，相信姐妹们都是 小能手啦！提高效率的工具体现了对时间的高效利用💻

Python 的前世今生 & 它的“超能力”！🦸‍♀️

话说 Python 这门语言，就像一个温柔又强大的小姐姐！她有很多优点，让人一用就爱上！😍

Python 的优点 ✨：

  1. 简单易学： 语法超级简洁清晰，就像和小姐妹聊天一样自然！再也不用对着那些天书般的代码发愁啦！📖➡️😊
  2. 可移植性强： 就像一个百搭单品，Python 代码几乎可以在各种操作系统上完美运行！一次编写，到处都能用！🌍
  3. 库超丰富： Python 拥有一个 বিশাল (巨大的意思！) 的“工具箱”，各种各样的库应有尽有，无论是想搞个网站、分析数据、还是玩玩人工智能，都能找到合适的“工具”！🧰
  4. 应用场景广泛： 简直是全能 ACE！Web 应用开发、大数据开发、数据分析、人工智能、桌面软件、运维测试… 哪里都能看到她的身影！简直是程序媛的百变战袍！👗

Python 的小缺点 🤫：

  • 执行效率稍慢： 解释型语言嘛，就像我们说话需要翻译一样，会稍微慢一点点。但对于我们日常应用来说，几乎感觉不到啦！🐌➡️🏃‍♀️

给你的代码加点“悄悄话”！💬

写代码的时候，为了方便自己和别人理解，我们经常会加一些“注释”，就像在聊天记录里加个表情或者备注一样！😉

  • 单行注释： 用 # 开头，这一行后面的内容 Python 就直接忽略啦！就像我们的小秘密，只有自己知道！🤫
  • 多行注释： 用三个双引号 """ 包裹起来，可以写很多很多行注释！就像我们的小日记本！📖

给你的数据找个“家”！变量和类型！🏠🏷️

在 Python 里，我们要存储各种各样的数据，就需要用到“变量”这个小容器！就像我们的化妆包，可以装各种不同的化妆品！💄 而且这个“化妆包”里的东西是可以随时更换的哦！😉

  • 变量： 用来存储数据的“小盒子”，里面的内容可以随时改变，但一次只能装一个“东西”哦！📦
  • 类型： 不同的数据有不同的“类型”，就像不同的化妆品有不同的功效一样！🧴
    • int：整数，就是没有小数点的数字啦！比如：1, 100, -5。
    • float：浮点数，就是带小数点的数字！比如：3.14, 1.68。
    • bool：布尔值，只有两个值：True（真）和 False（假），就像我们今天的心情！😊/😔
    • str：字符串，就是一串文本，用引号（单引号或双引号）包起来！比如：”Python”, ‘真香’。

给你的“宝贝”起个好听的名字！标识符和命名规范！🏷️

我们创建的变量、函数等等，都需要起个名字，这就是“标识符”！起个好名字就像给我们的小宠物起个可爱的昵称一样，方便我们记住和使用！🐶

命名规范小 Tips：

  1. 见名知意！ 起的名字要能一眼看出是干啥用的！比如存名字的变量就叫 name，存年龄的就叫 age，这样代码才更容易读懂！就像我们给文件命名一样，要清晰明了！📂
  2. 字母、数字、下划线组成： 我们的“昵称”只能包含这三种哦！不能有空格或者其他特殊符号！🚫
  3. 数字不能开头！ “昵称”不能以数字开头哦！🙅‍♀️
  4. 区分大小写！ myName 和 MyName 是两个不同的“昵称”哦！👀

让你的程序“说句话”！print 输出函数！🗣️

print() 是 Python 里最常用的一个“说话”的函数啦！它可以把你想展示的内容输出到屏幕上，就像我们发朋友圈一样！📱

姐妹们，记住这句 Python 界的金句：人生苦短，我用 Python！ 😎 学会 Python，解放你的双手，让生活更美好！快点加入 Python 的大家庭吧！💖✨
