06-软件安装和系统服务

Posted on 2025-04-26

兄弟们，今天来聊聊在 Linux 系统上怎么“捯饬”各种软件和系统服务！💻✨ 别看命令行黑乎乎的，玩熟了简直打开新世界的大门！😎

软件分类：你的 App “出生”方式大揭秘！👶➡️🚀

话说软件就像我们的小可爱 App，它们的“出生”方式可不一样哦！主要分两种：

1. 源码包： 就像拿到 App 的“源代码”！📜 你需要自己配置好“开发环境”（各种依赖），然后像个小码农一样“编译”它，再“打包”成能安装的“安装包”，最后才能安装使用。听起来是不是有点硬核？💪 但好处是你可以根据自己的需求定制功能，就像自己 DIY 一个独一无二的手机壳！🎨

举个栗子🌰： 很多超 cool、最新的开源软件都是以源码包的形式发布的。比如你想用某个还没被主流 Linux 发行版收录的黑科技工具，很可能就需要你先下载源码，然后 ./configure -> make -> make install 三连击！💥

2. 二进制包： 这就像直接拿到已经打包好的 App 安装包（比如安卓的 .apk，iOS 的 .ipa）。你不需要关心它背后的“源代码”是啥，直接“一键安装”就能用！✌️ 这种方式为了保护开发者的知识产权，会进行代码混淆，让别人不容易直接看到源码。

二进制包又分离线安装和在线安装两种方式，各有千秋，适用不同场景哦！

RPM 离线安装：网不好？没关系！📦

RPM（Red Hat Package Manager）是很多 Linux 发行版（比如 CentOS、Fedora）都在用的“快递”格式！📦 你可以提前下载好 .rpm 包，然后在没网或者网络不好的情况下直接安装！

常用命令小贴士：

  • 查询已安装的 RPM 包： rpm -qa （这个命令会列出所有已安装的 RPM 包，信息量有点大哈！👀）
  • 精准查找包含特定关键词的包： rpm -qa | grep <关键词> 比如你想找包含 wget 的包：rpm -qa | grep wget
  • 查看特定 RPM 包的详细信息： rpm -qi <包名> 比如：rpm -qi wget-1.21.1-8.el9.x86_64
  • 安装本地 RPM 包： rpm -ivh <.rpm 包文件> i 是安装，v 是显示详细信息，h 是显示安装进度小方块，敲起来是不是很有仪式感？🧱
    • 强制安装（小心使用！）： 有时候遇到依赖问题，可以尝试加上 --nodeps --force，但可能会导致软件运行不稳定哦！⚠️
  • 升级 RPM 包： rpm -Uvh <.rpm 包文件> U 是升级，如果没安装过就相当于安装。
  • 卸载 RPM 包： rpm -e <包名> 注意这里是包名，不是 .rpm 文件名！比如卸载 wget：rpm -e wget
    • 强制卸载（同样小心！）： 同样可以加上 --nodeps --force，但后果自负哦！💣

DNF 在线安装：冲浪达人的必备技能！🏄‍♀️

DNF（Dandified YUM）是新一代的包管理器，在 Fedora 和一些新的 CentOS 版本上取代了 YUM。它就像一个智能的“App Store”，连接着软件仓库，可以自动帮你下载、安装、升级、卸载软件，还能自动处理依赖关系，简直是懒人必备！😎

常用命令速记：

  • 列出所有可用的软件包： dnf list （信息量巨大！🐳）
  • 搜索软件包： dnf search <关键词> 比如你想找关于 Python 的包：dnf search python
  • 查看软件包信息： dnf info <包名> 比如：dnf info python3
  • 安装软件包： dnf install <包名> 比如安装 Python：dnf install python3 它会自动帮你下载并安装所有相关的依赖包，超省心！💖
  • 升级软件包： dnf upgrade <包名> 升级指定的包，比如升级 Python：dnf upgrade python3
  • 升级所有软件包： dnf update 或者 dnf upgrade （推荐定期执行，保持系统软件最新！✨）
  • 移除软件包： dnf remove <包名> 比如移除 Python：dnf remove python3
    • 移除不再需要的依赖包： dnf autoremove 卸载掉那些因为之前安装某个包而自动安装，但现在那个包已经被卸载了的“孤儿”依赖包，让你的系统更干净！🧹
  • 列出已安装的软件包： dnf list --installed 想看看自己都装了啥？用这个！🧐
  • 查找已安装的包含特定关键词的包： dnf list --installed | grep <关键词> 比如查找已安装的 Python 包：dnf list --installed | grep python
  • 清理缓存： dnf clean all 有时候遇到问题可以尝试清理一下缓存。

举几个 DNF 的实用例子🌰：

  • 想安装一个超好用的文本编辑器 nano： sudo dnf install nano
  • 发现 vim 太难学，想换回 nano？ sudo dnf remove vim
  • 听说 neofetch 可以酷炫地展示系统信息，想试试？ sudo dnf install neofetch 然后运行 neofetch 看看效果！🤩
  • 系统提示有更新，想一次性升级所有软件？ sudo dnf update -y （加上 -y 可以自动确认，不用每次都按 y 啦！😉）

好啦，关于 Linux 下的软件安装就先聊到这里！是不是感觉也没那么难嘛？😉 掌握了这些小技巧，你也能轻松管理你的 Linux 系统啦！下次再和兄弟们分享更多好玩的技术干货！💖✨
