✨ Python新手必看！Day05容器与函数基础知识点总结！ 🚀

哈喽兄弟们！今天给大家整理的是Python学习第五天的干货笔记，主要讲的是容器的一些通用方法和属性，以及超实用的函数哦！快拿出小本本记下来！📝

# 容器的通用方法

Python的列表、元组、字符串这些容器，有一些通用的方法可以大大提高我们的效率！

  • enumerate()：这个函数简直是遍历容器时的神器！它可以同时给出元素的索引和值，就像这样： 再也不用手动去数索引啦！🥳
  • len()：想知道容器里有多少个元素？用len()就对啦！
  • range()：生成一个数字序列，经常用在循环里！
  • max() 和 min()：分别用来找出容器中的最大值和最小值。

# 容器中通用的属性

容器还有一个很重要的特性，那就是可以通过索引来访问和操作元素！

  • 索引取值：就像数组一样，用方括号 [] 加上索引就可以获取特定位置的元素。记住索引是从0开始的哦！
  • 切片操作：想要一次获取容器中多个连续的元素？切片了解一下！
  • del 关键字：可以用来删除容器中的元素。 ⚠️ 注意：列表是可变的，所以删除操作会直接修改列表本身哦！

# 敲黑板！函数的定义与使用

函数是Python中非常重要的概念，它可以帮助我们组织代码，提高代码的复用性！

  • 定义函数：使用 def 关键字来定义一个函数，后面跟着函数名、括号 () 和冒号 :。函数体需要缩进。
  • 调用函数：直接使用函数名加上括号 () 就可以调用函数了。如果函数有参数，需要在括号里传入相应的值。
