03-控制流程

Posted on 2025-04-26

兄弟们，今天我们来聊聊 Python 里的“循环”小宇宙！🌌✨ 它能让我们的代码像上了发条的小马达一样，重复执行我们想要的操作，简直是解放双手、提升效率的神器！😎💖

循环中的“俄罗斯套娃”！嵌套循环！🎁🎁🎁

想象一下，循环里面又套着一个循环，就像我们小时候玩的俄罗斯套娃一样！一层又一层，可以实现一些更复杂的逻辑哦！

遍历小能手！for 循环！🚶‍♀️🚶🚶‍♂️

for 循环就像一个贴心的导游，可以带着我们一个一个地“参观”容器里的每个“景点”（元素）。容器可以是列表、字符串等等，只要里面有东西，for 循环就能把它遍历个遍！

for 循环还能指定循环次数哦！用 range() 函数就可以实现啦！就像我们跑步，可以指定跑几圈！🏃‍♀️

永不停歇的小马达！无限循环！<0xF0><0x9F><0xAA><0xB0>

有时候我们希望程序一直运行下去，直到我们手动停止它，这时候就可以用 while True: 制造一个无限循环！就像一个永动机！

关键时刻喊“停”！break！✋

break 就像我们生活中的“紧急停止”按钮！在循环进行的过程中，如果遇到了 break，程序会立刻跳出整个循环，后面的代码也不会再执行啦！

“跳过”这个！continue！➡️

continue 就有点像我们生活中的“跳过”！在循环进行的过程中，如果遇到了 continue，程序会跳过当前这一次循环剩下的代码，直接进入下一次循环！

字符串的“门牌号”！索引！<0xF0><0x9F><0x97><0x8E>️

字符串就像一串珠子，每个字符都有自己的“门牌号”，也就是索引！索引从 0 开始计数哦！我们还可以用负数索引从末尾开始计数，-1 代表最后一个字符！

“切”出一块美味！切片！🔪🍰

切片就像我们切蛋糕一样，可以从字符串中“切”出一小块！用 : 来指定切片的范围！

字符串的“魔法棒”！常用操作！✨🪄

字符串还有很多自带的“魔法棒”，可以帮我们查找、修改、分割字符串！

  • 查找：find() 和 index()

  • 替换：replace()

  • 分割：split()

好啦兄弟们，今天的 Python 控制流程就聊到这里啦！掌握了循环和字符串的各种操作，你的 Python 代码就能更加灵活、更加强大啦！💪 快去试试这些“魔法”，让你的程序动起来吧！💖✨
